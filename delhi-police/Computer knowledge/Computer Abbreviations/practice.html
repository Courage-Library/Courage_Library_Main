Delhi Police Exam

Practice Questions

1

What is the full form of HTML?

Options: (a) HyperText Transfer Link (b) HyperText Markup Language (c) HyperText Management Layer (d) Hyper Transfer Markup Language

View Answer

HyperText Markup Language

2

Which protocol is used for secure web browsing?

Options: (a) HTTP (b) HTTPS (c) FTP (d) SMTP

View Answer

HTTPS

3

What does URL stand for?

Options: (a) Uniform Resource Locator (b) Universal Resource Link (c) Uniform Request Location (d) Universal Retrieval Link

View Answer

Uniform Resource Locator

4

Which of the following is used to send emails?

Options: (a) FTP (b) SMTP (c) POP3 (d) DNS

View Answer

SMTP

5

DNS is used to:

Options: (a) Assign IP addresses (b) Translate domain names to IP addresses (c) Encrypt data (d) Store web pages

View Answer

Translate domain names to IP addresses

6

PDF stands for:

Options: (a) Program Data File (b) Portable Document Format (c) Personal Data Folder (d) Private Document File

View Answer

Portable Document Format

7

Which protocol transfers files between computers?

Options: (a) HTTP (b) FTP (c) DNS (d) SMTP

View Answer

FTP

8

IMAP protocol is used for:

Options: (a) Sending emails (b) Receiving emails while keeping them on the server (c) Encrypting emails (d) Accessing websites

View Answer

9

WWW stands for:

Options: (a) World Wide Web (b) Web Wide World (c) Wide World Web (d) World Web Wide

World Wide Web

10

VPN is used for:

Options: (a) Faster internet speed (b) Securing internet connection (c) Transferring files (d) Sending emails

Securing internet connection

11

HTTP is used to:

Options: (a) Access email servers (b) Transfer web pages (c) Assign IP addresses (d) Encrypt data

Transfer web pages

IP in networking stands for:

13

Which of these converts domain names to numbers?

Options: (a) DNS (b) HTTP (c) FTP (d) ISP

DNS

POP3 is used for:

Options: (a) Sending emails (b) Receiving emails (c) Browsing websites (d) Transferring files

Practice Assignment

Task: Create a "Software & Internet Acronym Sheet"

Create a comprehensive reference sheet of important software and internet acronyms.

1

Open MS Word / Excel

2

Create a 2-column table:

  • Column 1 → Acronym (e.g., HTML, HTTP, URL, PDF, SMTP)
  • Column 2 → Full Form + One-line Use
3

Apply bold to acronyms and blue font color to full forms

4

Save the file as Software_Internet_Acronyms.docx

Pro Tip: Revise the sheet daily. Try making a mini quiz for yourself — hide answers, time yourself for 10 seconds per question.

Memory Tricks & Quick Recap

H in HTML/HTTP → Web-related / HyperText

P in SMTP/FTP → Protocol / Process

U in URL → Uniform / Universal Locator

HTTPS → Always remember "S = Secure"

DNS → Think Name → Number translation

Flashcards or tables work best for acronyms memorization

Exam Tip: These 20 acronyms and their uses are highly likely to appear in Delhi Police Constable / Sub-Inspector computer knowledge section. Practice regularly and revise before the exam!
