Computer Data Measurement

Delhi Police Exam

1. What is Computer Data Measurement?

All data in a computer is stored or processed in the form of binary digits (0 and 1), known as bits. To make calculations easier, bits are grouped into larger units such as bytes, kilobytes, megabytes, gigabytes, and terabytes.

2. Basic Units of Measurement

Unit Full Form Equivalent Value Explanation / Use
Bit Binary Digit Smallest unit of data Represents 0 or 1 only
Byte 8 Bits 8 bits = 1 byte Can store 1 character (like 'A')
Kilobyte (KB) 1 KB = 1024 Bytes ≈ 1 small text page Used in documents
Megabyte (MB) 1 MB = 1024 KB ≈ 1 photo / short song Used in media files
Gigabyte (GB) 1 GB = 1024 MB ≈ 1 movie or 1000 photos Used in smartphones / RAM
Terabyte (TB) 1 TB = 1024 GB ≈ 1000 movies Used in hard drives / servers

Bit → Byte → Kilobyte (KB) → Megabyte (MB) → Gigabyte (GB) → Terabyte (TB) → Petabyte (PB)

4. Important Concepts

1 byte = 8 bits

1 KB = 1024 bytes, not 1000 bytes (because computers work in powers of 2)

Data Storage refers to how much data a device can hold

Data Transfer Speed is measured in bits per second (bps), kbps, Mbps, or Gbps
