Internet & Email Practice Questions

Q1. What is the Internet?

View Answer

The Internet is a global network that connects millions of computers for sharing data, information, and communication.

Q2. What does ISP stand for?

View Answer

Internet Service Provider â€“ a company that provides internet access to users (e.g., Jio, Airtel, BSNL).

Q3. What is an IP Address?

View Answer

A unique numerical label assigned to each device connected to the internet, like a digital address.

Q4. What does "URL" stand for?

View Answer

Uniform Resource Locator â€“ the web address used to access websites (e.g., https://www.google.com).

Q5. What is a Domain Name?

View Answer

The readable name of a website that represents its IP address (e.g., www.yahoo.com).

Q6. Which of the following is NOT a search engine?

View Answer

Microsoft Edge (it is a browser, not a search engine).

Q7. Name any two popular search engines.

View Answer

Google and Bing.

Q8. What is a Web Browser?

View Answer

A software used to access, view, and navigate websites on the internet (e.g., Chrome, Firefox, Safari).

Q9. What is the function of the "Refresh" button in a browser?

View Answer

It reloads the current web page to show updated content.

Q10. In email communication, what is the purpose of CC and BCC?

View Answer

CC (Carbon Copy) sends a visible copy to others, while BCC (Blind Carbon Copy) sends a hidden copy.

Q11. What is the "Inbox" in an email account used for?

View Answer

To store all received emails.

Q12. What does the term "Spam" refer to in email?

View Answer

Unwanted or junk emails, often promotional or fraudulent.

Q13. What is a "Draft" email?

View Answer

An email that has been written but not yet sent.

Q14. What is "Netiquette"?

View Answer

The set of polite and respectful rules for communicating and behaving properly on the internet.

Q15. Which of the following is an example of bad netiquette?

View Answer

Writing messages in ALL CAPITAL LETTERS (considered shouting online).
