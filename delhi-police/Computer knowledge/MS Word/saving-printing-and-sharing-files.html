Saving, Printing, and Sharing Files

Delhi Police Exam

1. Saving Files

Saving files ensures your work is stored safely and can be reopened later.

Key Options:

Save (Ctrl + S)

Saves changes to an existing file

Save As (Ctrl + Shift + S)

Save document in different location or format (.docx, .pdf, .txt)

AutoSave

Available in Office 365, automatically saves document in OneDrive

Exam Tip: Ctrl + S and Ctrl + Shift + S are frequently asked in Delhi Police PYQs.

2. Printing Documents

Printing prepares your document for physical copy.

Steps to Print:

1

File → Print or Ctrl + P

2

Select Printer → Choose available printer

3

Pages → All pages / Specific pages

4

Copies → Number of copies

5

Print Layout & Options → Portrait/Landscape, Collated/Uncollated

Shortcut Tip: Ctrl + P → Quick access to print dialog

3. Sharing Documents

Documents can be shared digitally with others.

Methods:

Email Attachment

File → Share → Email

Cloud Sharing

Upload to OneDrive or Google Drive → Share link

Collaboration

Real-time editing via Office 365 or Google Docs

Exam Note: Delhi Police often asks about methods of sharing, especially via Email and Cloud.

5. Quick Recap

Saving: Ctrl + S (Save), Ctrl + Shift + S (Save As), AutoSave in OneDrive/Office 365

Printing: Ctrl + P → Select Printer → Pages → Copies → Layout options

Sharing: Email attachment, Cloud sharing (OneDrive, Google Drive), Real-time collaboration

Important Shortcuts: Ctrl+S, Ctrl+Shift+S, Ctrl+P

Tip: These topics are highly frequent in Delhi Police and SSC computer exams.

You've completed File Operations in MS Word!
