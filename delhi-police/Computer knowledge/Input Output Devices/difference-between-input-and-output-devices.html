Difference Between Input and Output Devices

Delhi Police Exams

Understanding the difference between input and output devices is a must for Delhi Police Computer Awareness exams. These questions frequently appear as one-liners, match the following, or MCQs. Here's a complete exam-ready article with tricks, PYQs, practice, and recap.

Key Differences

Feature Input Devices Output Devices
Definition Devices used to send data to the computer for processing Devices used to receive or display processed data from the computer
Function Data entry / command input Data presentation / result output
Examples Keyboard, Mouse, Scanner, MICR, OMR, OCR Monitor, Printer, Plotter, Speaker, Projector
Direction of Data User → CPU CPU → User
Type of Output — Visual, Audio, Hardcopy

Tip: Remember the data flow direction - Input devices send data TO computer, Output devices receive data FROM computer

Quick Recap & Tricks

Device Categories:

Input devices: Keyboard, Mouse, Scanner, MICR, OMR, OCR → User to CPU

Output devices: Monitor, Printer, Plotter, Speaker, Projector → CPU to User

Both: Touchscreen, Interactive Whiteboards

Memory Trick:

"In Out Both – Keyboard Mouse Scanner Speak Plot Touch"

Input → Output → Both
Keyboard → Mouse → Scanner → Speaker → Plotter → Touchscreen
