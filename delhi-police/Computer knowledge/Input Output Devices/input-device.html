Input Devices

Delhi Police Exams

In the Delhi Police Computer Knowledge section, input devices are frequently asked in MCQs, especially in 1-mark questions. Understanding each device, its use, and type will help you answer quickly. Here's a complete exam-ready article with tricks, PYQs, practice, and recap.

Common Input Devices

Device Full Form / Meaning Use / Function
Keyboard — Used to enter text and commands into the computer
Mouse — Pointing device to select, drag, and click on-screen objects
Scanner — Converts physical documents/images into digital format
MICR Magnetic Ink Character Recognition Reads characters printed with magnetic ink (used in banking cheques)
OMR Optical Mark Recognition Reads marks on specially designed forms (e.g., multiple-choice answer sheets)
OCR Optical Character Recognition Converts printed or handwritten text into digital text
Trackball — Stationary pointing device; user rotates ball to move cursor
Joystick — Controls movement in games or simulations
Digital Pen / Stylus — Input device for writing/drawing on touch screens
Webcam — Captures video or images for input

Tip: Focus on understanding the specific use cases for each input device, especially MICR, OMR, and OCR

MICR → Banking, magnetic ink
