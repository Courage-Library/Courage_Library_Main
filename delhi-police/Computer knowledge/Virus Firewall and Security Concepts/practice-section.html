1

Which of the following is considered a strong password?

Options: (a) 123456 (b) Password@2025 (c) admin (d) qwerty

View Answer

Password@2025

2

A firewall is mainly used to:

Options: (a) Delete files (b) Prevent unauthorized network access (c) Increase system speed (d) Format hard disk

View Answer

Prevent unauthorized network access

3

Phishing attacks aim to:

Options: (a) Steal sensitive information (b) Print documents (c) Encrypt files (d) Speed up the network

View Answer

Steal sensitive information

4

Which of these is an input device?

Options: (a) Monitor (b) Keyboard (c) Printer (d) Speaker

View Answer

Keyboard

5

Which malware type encrypts user files and demands ransom?

Options: (a) Trojan (b) Virus (c) Ransomware (d) Worm

View Answer

Ransomware

6

HTTPS in a website URL indicates:

Options: (a) Insecure connection (b) Secure and encrypted connection (c) Paid website (d) Outdated browser

View Answer

Secure and encrypted connection

7

Which of the following is a software firewall?

Options: (a) Router (b) Windows Firewall (c) Hardware Firewall (d) Switch

View Answer

Windows Firewall

8

NGFW stands for:

Options: (a) Network General Firewall (b) Next-Generation Firewall (c) New Gateway Firewall (d) Network Gateway Fast

View Answer

Next-Generation Firewall

9

The first generation of computers used:

Options: (a) Transistors (b) Vacuum tubes (c) Integrated Circuits (d) Microprocessors

View Answer

Vacuum tubes

10

Which of the following is not a type of malware?

Options: (a) Worm (b) VPN (c) Trojan (d) Virus

View Answer

VPN

11

Which of these is used to scan printed text into a digital form?

Options: (a) MICR (b) OCR (c) Projector (d) Speaker

View Answer

OCR

12

1 GB equals how many MB (binary)?

Options: (a) 1000 MB (b) 1024 MB (c) 500 MB (d) 2048 MB

View Answer

1024 MB

13

Which shortcut is used to paste in MS Word?

Options: (a) Ctrl + C (b) Ctrl + V (c) Ctrl + X (d) Ctrl + P

View Answer

Ctrl + V

14

Which type of printer is commonly used for high-quality images?

Options: (a) Dot Matrix (b) Inkjet (c) Line Printer (d) Thermal

View Answer

Inkjet

15

The CPU is also known as:

Options: (a) Hard drive (b) Brain of the computer (c) Output device (d) Input device

View Answer

Brain of the computer

16

Which antivirus software is free for basic use?

Options: (a) Kaspersky (b) Avast (c) Norton (d) McAfee

View Answer

Avast

17

Two-factor authentication (2FA) is used to:

Options: (a) Add extra security (b) Format system (c) Speed up browser (d) Encrypt files automatically

View Answer

Add extra security

18

Which type of firewall acts as an intermediary between user and internet?

Options: (a) Packet Filtering (b) Proxy (c) Hardware (d) NGFW

View Answer

Proxy

19

What does HTTPS stand for?

Options: (a) HyperText Transfer Protocol Secure (b) HyperText Transfer Protocol Simple (c) High Transfer Protocol System (d) HyperText Transmission Private System

View Answer

HyperText Transfer Protocol Secure

20

Safe browsing practice includes:

Options: (a) Clicking all links (b) Using strong passwords and verifying URLs (c) Downloading from untrusted sites (d) Sharing passwords publicly

View Answer

Using strong passwords and verifying URLs
