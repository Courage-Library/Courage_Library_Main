Basic Windows Commands

Delhi Police Exams

Understanding basic Windows commands is essential for Delhi Police Computer Awareness exams. Questions often test command usage, shortcuts, and functions. Here's a complete exam-ready article with tricks, PYQs, practice, and recap.

Overview of Basic Windows Commands

1

Run Command

  • Opens programs, folders, documents, or internet resources quickly
  • Shortcut: Windows + R
  • Examples: cmd → Command Prompt, notepad → Notepad

Quick Access: Use Run dialog for fast program launching

2

Control Panel

  • Configure hardware, software, system settings, and user accounts
  • Shortcut: Windows + X → Control Panel / Settings
  • Manages system configuration and preferences

System Management: Central hub for Windows settings

3

File Explorer

  • Access files, folders, and drives efficiently
  • Shortcut: Windows + E
  • Features: Search files, copy/move, create folders

File Management: Primary tool for organizing files

4

Task Manager

  • Monitors running applications, processes, and system performance
  • Shortcut: Ctrl + Shift + Esc or Ctrl + Alt + Del → Task Manager
  • Shows CPU, memory, disk, and network usage

Performance Monitor: Essential for troubleshooting

5

Command Prompt (CMD)

  • Execute commands to control files, system settings, and network tasks
  • Shortcut: Windows + R → type cmd
  • Advanced system administration tool

Power User Tool: For advanced Windows operations

6

Run Command Tricks

  • msconfig → System configuration
  • regedit → Registry Editor
  • services.msc → Services management
  • calc → Calculator
  • mspaint → Paint application

Quick Launch: Memorize these for faster system access

Practice Questions (Delhi Police PYQ-style)

1

Which shortcut opens the Run dialog box?

Options: (a) Ctrl + R (b) Windows + R (c) Alt + R (d) Windows + E

Category: Shortcut Keys

Show Answer

Windows + R

2

Which command is used to open the Command Prompt?

Options: (a) notepad (b) cmd (c) explorer (d) msconfig

Category: Run Commands

Show Answer

cmd

3

File Explorer can be opened by which shortcut?

Options: (a) Windows + F (b) Windows + E (c) Ctrl + E (d) Alt + F

Category: Shortcut Keys

Show Answer

Windows + E
