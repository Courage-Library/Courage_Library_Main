AVERAGES (Simple Average)

SSC GD Exam Preparation - Quantitative Aptitude

üîπ 1. What is Average?

Average means the equal distribution of a total quantity among a set of items or persons.

Average = Sum of values / Number of values

üîπ 2. Important Concepts

‚úî Sum = Average √ó Number of items

This is the most fundamental relationship in average problems.

‚úî If one value is replaced:

New Sum = Old Sum - Removed Value + Added Value

‚úî If the average increases or decreases:

Increase in Total = Change in Average √ó Number of Items

Formula 2 ‚Äî Weighted Total

Total = Average √ó Number

Formula 3 ‚Äî Finding Missing Number

Missing Value = Total Required - Total of Known Values

üîπ 4. Examples

Example 1

Numbers: 5, 7, 9

Average = (5 + 7 + 9)/3 = 7

Example 2

Average of 6 numbers = 12

Total = 6 √ó 12 = 72

Example 3

Average of 5 numbers is 20. If one number (30) is removed:

New sum = 100 ‚Äì 30 = 70
New average = 70/4 = 17.5
