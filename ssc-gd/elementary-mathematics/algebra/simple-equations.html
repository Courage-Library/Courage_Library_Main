Algebra deals with numbers and variables. A simple equation is an equation that contains one variable and can be solved using basic arithmetic operations.

General Form:

ax + b = c

x = unknown variable

a, b, c = constants

Step 1: Move constants to the other side

ax = c - b

Step 2: Divide by coefficient of x

x = (c - b) / a

Example:

Solve 3x + 5 = 20

3x = 20 - 5 = 15

x = 15 √∑ 3 = 5
