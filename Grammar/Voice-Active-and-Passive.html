Voice (Active–Passive) for SSC CGL

Complete Grammar Guide

1. What is Voice?

Voice shows whether the subject performs the action or receives it.

Active Voice

Subject does the action

Example: Ravi eats an apple

Passive Voice

Subject receives the action

Example: An apple is eaten by Ravi

2. Basic Rules for Transformation

Rule 1: Tense Changes in Passive

Active Passive
Present Simple: S + V1(s/es) + O O + is/are + V3 + by S
Present Continuous: S + is/are + V-ing + O O + is/are being + V3 + by S
Present Perfect: S + has/have + V3 + O O + has/have been + V3 + by S
Past Simple: S + V2 + O O + was/were + V3 + by S
Past Continuous: S + was/were + V-ing + O O + was/were being + V3 + by S
Past Perfect: S + had + V3 + O O + had been + V3 + by S
Future Simple: S + will + V1 + O O + will be + V3 + by S
Future Continuous: S + will be + V-ing + O Rarely used; focus on SSC pattern
Future Perfect: S + will have + V3 + O O + will have been + V3 + by S

Rule 2: Object Becomes Subject

The object of active voice becomes the subject of passive

Active → Ram reads a book

Passive → A book is read by Ram

Rule 3: Subject Becomes Agent (optional)

  • Use "by + subject" in passive
  • He is respected by everyone
  • Agent can be omitted if obvious
  • The cake was eaten

Rule 4: Special Cases

1. Imperative Sentences

Active → Close the door

Passive → Let the door be closed

2. Modal Verbs

Active → She can solve the problem

Passive → The problem can be solved by her

3. Two Objects

Indirect object becomes subject:

Active → Ram gave Sita a gift

Passive → Sita was given a gift by Ram

Direct object becomes subject (optional, less common):

A gift was given to Sita by Ram

4. Questions

Active → Who wrote this book?

Passive → By whom was this book written?

3. Common SSC Error Patterns

1. Forgetting "be" in passive

The work done by Ram

The work is done by Ram

2. Wrong tense transformation

He eats an apple → An apple was eaten by him

Correct → An apple is eaten by him

3. Misplacing agent

The book is read Ram

The book is read by Ram

4. Modal verbs mistake

She must solve the problem → The problem must be solve by her

The problem must be solved by her

4. SSC CGL Previous Year Questions

Q1. Ram wrote a letter. (Change to Passive)

Answer: A letter was written by Ram

Q2. Who wrote this book? (Passive)

Answer: By whom was this book written?

Q3. She can solve the problem. (Passive)

Answer: The problem can be solved by her

Q4. Close the door. (Passive)

Answer: Let the door be closed

Q5. Ram gave Sita a gift. (Passive)

Answer: Sita was given a gift by Ram

5. Quick Recap

  • Object → Subject, Subject → Agent (optional)
  • Check tense and verb form
  • Use by for agent (optional if obvious)
  • Watch modals, imperatives, questions
  • SSC traps: wrong tense, missing "be", incorrect participle
