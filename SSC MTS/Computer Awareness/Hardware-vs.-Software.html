Hardware vs Software

SSC MTS Exam

1. What is Hardware?

Hardware refers to the physical parts of a computer system — the components that you can see and touch. They form the body of the computer and help in processing data.

Physical & Tangible Components

Can be Seen and Touched

Forms Body of Computer

Helps in Processing Data

Simple Definition: Hardware is the collection of tangible components of a computer that perform operations and store data.

Types of Hardware

Examples of Hardware

Keyboard

Mouse

Monitor

Printer

CPU

RAM

Hard Disk

Motherboard

Hardware Categories

Category Description Examples
Input Devices Used to enter data Keyboard, Mouse, Scanner
Processing Devices Perform operations CPU (ALU + CU + Registers)
Storage Devices Store data permanently or temporarily Hard Disk, Pen Drive, RAM
Output Devices Display results Monitor, Printer, Speaker
Communication Devices Help in data transfer Modem, Router, Wi-Fi Adapter

2. What is Software?

Software is a set of programs, instructions, and data that tells the hardware what to do. It forms the intelligence or brain of the computer.

Set of Programs & Instructions

Non-Physical & Intangible

Forms Intelligence of Computer

Tells Hardware What to Do

Simple Definition: Software is a collection of programs that enables a computer to perform specific tasks.

Types of Software

Examples of Software

MS Word

Excel

Windows

Linux

Chrome

Photoshop

VLC Player

Android

Software Categories

Type Description Examples
System Software Controls hardware and runs the system Windows, Linux, macOS
Application Software Performs specific user tasks MS Office, Browsers, Games
Utility Software Helps in maintenance and protection Antivirus, File Manager, WinZip

3. Difference Between Hardware and Software

Basis Hardware Software
Nature Physical & tangible Non-physical & intangible
Function Executes operations Provides instructions
Failure Can wear out or damage Corrupts or crashes
Development Made of electronic components Written using programming languages
Examples CPU, Keyboard, Monitor Windows, Word, Photoshop

4. Real-Life Analogy

Think of a computer like a human body:

Hardware

→ Body parts

(eyes, hands, brain hardware)

Software

→ Mind & instructions

(how to use the body)

Without software, hardware is just a dead machine; without hardware, software can't function.

5. Common Delhi Police / SSC Exam Facts

Hardware = Physical part; Software = Set of instructions

MS Word, Excel → Application Software

Windows, Linux → System Software

Antivirus → Utility Software

CPU → Hardware, not software

Software needs hardware to function

6. Quick Recap

Concept Summary
Hardware Tangible components like CPU, Monitor, Keyboard
Software Intangible programs that control hardware
System Software OS that runs the system
Application Software Used for specific user tasks
Utility Software Maintains system health
