Introduction to Computers

SSC MTS Exams

1. What is a Computer?

A computer is an electronic device that processes data according to a set of instructions to produce meaningful information. It can store, process, and retrieve data quickly and accurately.

Electronic Device

Processes Data

Follows Instructions

Produces Information

Simple Definition: A computer is an electronic machine that takes input, processes it, and gives output.

IPO Cycle - Fundamental Concept

Input → Process → Output

Input

Data given to computer

Keyboard, Mouse, Scanner

Process

Converting data to information

CPU Operations

Output

Final result displayed

Monitor, Printer, Speaker

Trick to Remember: I-P-O Rule → Input → Process → Output

2. Characteristics of a Computer

Feature Description
Speed Performs tasks much faster than humans
Accuracy Very few chances of errors
Storage Can store a large amount of data
Automation Works automatically after giving instructions
Diligence Never gets tired or bored
Versatility Can perform multiple tasks
Connectivity Can connect globally through networks

Trick: SAS-DVC → Speed, Accuracy, Storage, Diligence, Versatility, Connectivity

3. Basic Terms in Computers

Term Meaning
Data Raw facts and figures
Information Processed and meaningful data
Input Data given to the computer
Processing Converting data into useful information
Output Final result displayed to the user
Storage Saving data permanently or temporarily

Example: Typing marks in MS Excel → Data
After calculating total marks → Information

4. Functional Units of Computer

Unit Description Examples
Input Unit Feeds data to computer Keyboard, Mouse
Processing Unit (CPU) Brain of computer ALU, CU, Registers
Memory / Storage Unit Saves data & programs RAM, Hard Disk
Output Unit Shows results Monitor, Printer

Trick: I – P – O – S → Input, Process, Output, Storage

5. Types of Computers

Type Description Examples
Microcomputer For personal use Laptop, Desktop
Minicomputer For small businesses Server systems
Mainframe Handles huge data Banking systems
Supercomputer Highest speed & power PARAM, Cray

Trick: M³S → Micro, Mini, Mainframe, Supercomputer

6. Components of a Computer System

Component Description
Hardware Physical parts like monitor, CPU, keyboard
Software Set of programs/instructions
User Person operating the computer
Data Raw input for processing

Trick: H + S + U + D → Hardware, Software, User, Data

7. Applications of Computers

Field Usage
Education Online learning, exams
Banking ATM, online transactions
Business Payroll, billing
Medicine Diagnosis, research
Government Record keeping, e-Governance
Defence Radar, missile control

Trick to Remember Fields: E-B-B-M-G-D → Education, Banking, Business, Medicine, Government, Defence

10. Common SSC & Delhi Police Exam Facts

CPU → Brain of Computer

Supercomputer of India → PARAM

Concept Summary
