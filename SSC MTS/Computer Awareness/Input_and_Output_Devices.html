Input & Output Devices

SSC MTS Exam

1. What are Input & Output Devices?

Input Devices are used to enter data and commands into the computer, while Output Devices are used to display or present information from the computer to the user.

Input → Data Entry

Output → Information Display

Two-way Communication

Essential for Computer Operation

Simple Definition: Input devices send data to computer, Output devices receive data from computer.

2. Input Devices

Common Input Devices

Keyboard

Mouse

Scanner

Microphone

Webcam

Joystick

Touchscreen

Barcode Reader

Types of Input Devices

Category Description Examples
Keyboard Devices Enter text and commands Keyboard, Keypad
Pointing Devices Control cursor movement Mouse, Trackball, Touchpad
Scanning Devices Convert physical to digital Scanner, Barcode Reader
Audio Input Capture sound Microphone, Voice Recorder
Video Input Capture images/video Webcam, Digital Camera
Touch Input Direct screen interaction Touchscreen, Graphics Tablet

3. Output Devices

Common Output Devices

Monitor

Printer

Speaker

Headphones

Projector

Plotter

Braille Reader

LED Indicators

Types of Output Devices

Category Description Examples
Visual Output Display visual information Monitor, Projector, LED Screen
Audio Output Produce sound Speaker, Headphones, Buzzer
Print Output Produce physical copies Printer, Plotter, 3D Printer
Tactile Output Provide physical feedback Braille Reader, Haptic Devices
Indicator Output Show status information LED Lights, LCD Displays

4. Difference Between Input & Output Devices

Basis Input Devices Output Devices
Function Send data to computer Receive data from computer
Data Flow From user to computer From computer to user
Purpose Enter commands and data Display results and information
Examples Keyboard, Mouse, Scanner Monitor, Printer, Speaker
Communication Input to processing unit Output from processing unit

5. Input/Output (I/O) Devices

Devices that perform both Input and Output

Touchscreen Devices

Input Function

Touch input for commands

Output Function

Visual display of information

Examples

Smartphones, Tablets, ATMs

Other I/O Devices

Modems

Send and receive data over networks

Network Cards

Transmit and receive network data

Touchpads

Input touch, output cursor movement

6. Real-Life Analogy

Think of a Restaurant Kitchen:

Input Devices

→ Order taking

(Waiters taking customer orders)

Output Devices

→ Serving food

(Waiters serving prepared dishes)

Input devices bring information in (like orders), Output devices send information out (like serving food).

7. Common Delhi Police / SSC Exam Facts

Keyboard & Mouse → Input Devices

Monitor & Printer → Output Devices

Touchscreen → Both Input & Output

Scanner → Input Device only

Speaker → Output Device only

Modem → Both Input & Output

8. Quick Recap

Device Type Function Key Examples
Input Devices Send data to computer Keyboard, Mouse, Scanner
Output Devices Receive data from computer Monitor, Printer, Speaker
I/O Devices Both input and output Touchscreen, Modem
Remember Input = Data IN, Output = Data OUT

You've completed Input & Output Devices!

Courage Tip: Remember that input devices bring data INTO the computer, while output devices send data OUT of the computer. Touchscreen devices are both input and output.
