Internet Basics & Web Browsers

SSC MTS Exam

1. What is the Internet?

The Internet is a global network of interconnected computers that communicate using standardized protocols. It is often called the "Network of Networks" because it connects millions of smaller networks worldwide.

Global Network

TCP/IP Protocols

Information Sharing

Decentralized System

Simple Definition: The Internet is a worldwide system of computer networks that allows users to share information and communicate.

6. URL Structure

Understanding a Web Address

https:// www.example.com :443 /path/to/page.html ?query=string #section

Protocol (https://)

→ How data is transferred

(Secure HTTP protocol)

Domain (www.example.com)

→ Website address

(Human-readable IP address)

A URL (Uniform Resource Locator) is the complete web address that tells your browser exactly where to find a specific resource on the internet.
