Number System

Essential Concepts for Delhi Police & SSC Exams

Introduction

Numbers are the foundation of arithmetic and every quantitative aptitude topic. The number system is a way of representing and classifying numbers based on their properties and usage.

Classification of Numbers

Numbers are broadly divided into several categories:

1. Natural Numbers (N)

The numbers used for counting are called Natural Numbers. They start from 1 and go up to infinity.

Examples:

1, 2, 3, 4, 5, 6, …

Set Representation:

N = {1, 2, 3, 4, 5, …}

Key Points:

  • The smallest natural number is 1
  • There is no largest natural number
  • Used for counting real-world objects like people, books, etc.

2. Whole Numbers (W)

When we include 0 with natural numbers, we get Whole Numbers. Hence, Whole Numbers = Natural Numbers + 0

Examples:

0, 1, 2, 3, 4, 5, 6, …

Set Representation:

W = {0, 1, 2, 3, 4, …}

Key Points:

  • The smallest whole number is 0
  • There is no largest whole number
  • Every natural number is a whole number, but 0 is not a natural number

3. Integers (Z)

Integers include all whole numbers and their negative counterparts. They can be positive, negative, or zero.

Examples:

-4, -3, -2, -1, 0, 1, 2, 3, 4 …

Set Representation:

Z = {…, -3, -2, -1, 0, 1, 2, 3, …}

Key Points:

  • 0 is neither positive nor negative
  • The set of integers is infinite in both directions
  • Every whole number and natural number is an integer

4. Rational Numbers (Q)

A number is said to be rational if it can be expressed in the form p/q, where p and q are integers and q ≠ 0.

Examples:

1/2, -3/4, 5, 0.25 (since 0.25 = ¼), -2 (since -2 = -2/1)

Set Representation:

Q = {p/q | p, q ∈ Z, q ≠ 0}

Key Points:

  • Every integer and terminating or repeating decimal is a rational number
  • 0.75 = ¾ → rational
  • 0.3333… = ⅓ → rational
  • All fractions are rational numbers

5. Irrational Numbers

Numbers that cannot be expressed in the form p/q are called irrational numbers. Their decimal expansions are non-terminating and non-repeating.

Examples:

√2, √3, √5, π, e, 0.101001000100001…

Properties:

Non-terminating, non-repeating decimals

Key Points:

  • √2 = 1.4142135… (never ends, never repeats)
  • π = 3.1415926…
  • Sum of a rational and irrational number is always irrational
  • Product of a non-zero rational and irrational number is always irrational

6. Real Numbers (R)

All rational and irrational numbers together form the set of Real Numbers.

Examples:

-3, 0, ½, √2, π

Set Representation:

R = Q ∪ Irrational Numbers

Key Points:

  • Every real number can be represented on a number line
  • Real Numbers include all numbers that are not imaginary
  • R = Q ∪ Irrational Numbers

7. Imaginary Numbers (Complex Numbers)

Numbers that involve the square root of a negative number are called imaginary numbers. They are written using i, where i = √(-1).

Examples:

i, 2i, -3 + 2i, etc.

Properties:

Cannot be placed on real number line

Key Points:

  • Imaginary numbers cannot be placed on the real number line
  • Complex numbers are formed by combining real and imaginary parts
  • Example: 2 + 3i

Special Types of Numbers

Even Numbers

Numbers divisible by 2

Examples: 0, 2, 4, 6, 8, 10, …

Note: 0 is an even number

Odd Numbers

Numbers not divisible by 2

Examples: 1, 3, 5, 7, 9, …

Prime Numbers

Numbers with exactly two distinct factors — 1 and itself

Examples: 2, 3, 5, 7, 11, 13, 17, …

2 is the only even prime number. 1 is not prime.

Composite Numbers

Numbers having more than two factors

Examples: 4, 6, 8, 9, 10, 12, …

Note: 1 is neither prime nor composite

Co-prime Numbers

Two numbers with HCF = 1

Examples: (4, 9), (5, 8), (7, 20)

They may or may not be prime themselves

Perfect Numbers

Sum of proper divisors equals the number

Examples: 6, 28

6 → 1+2+3=6, 28 → 1+2+4+7+14=28

Twin Primes

Prime numbers that differ by 2

Examples: (3, 5), (5, 7), (11, 13)

Consecutive Numbers

Numbers that follow each other without gaps

Examples: 1, 2, 3, 4 or 10, 11, 12

For any 3 consecutive integers → one divisible by 3

Triangular Numbers

Numbers forming equilateral triangle with dots

Formula: n(n + 1)/2

Examples: 1, 3, 6, 10, 15, 21, 28, …

Perfect Squares & Cubes

Squares and cubes of integers

Squares: 1, 4, 9, 16, 25, …

Cubes: 1, 8, 27, 64, 125, …

Key Relationships Between Sets

Category Symbol Contains
Natural Numbers N {1, 2, 3, …}
Whole Numbers W {0, 1, 2, 3, …}
Integers Z {…, -3, -2, -1, 0, 1, 2, …}
Rational Numbers Q p/q where q ≠ 0
Irrational Numbers — Non-repeating, non-terminating decimals
Real Numbers R All Rational + Irrational

Hierarchy:

N ⊂ W ⊂ Z ⊂ Q ⊂ R

Natural Numbers are a subset of Whole Numbers, which are a subset of Integers, and so on.
