1. Algebraic Simplification

Definition

Algebraic simplification means reducing an algebraic expression to its simplest form by using basic algebraic identities and operations.

Basic Concepts

Algebraic Expression

Combination of constants, variables, and mathematical operators (+, −, ×, ÷).

Example: 3x + 2y − 5

Like Terms

Terms with the same variables and powers.

Example: 3x, −5x are like terms.

Unlike Terms

Terms with different variables or powers.

Example: 2x, 3y are unlike terms.

Common Algebraic Identities

(a + b)² = a² + 2ab + b²

(a - b)² = a² - 2ab + b²

(a + b)(a - b) = a² - b²

(a + b + c)² = a² + b² + c² + 2(ab + bc + ca)

(a - b)³ = a³ - b³ - 3ab(a - b)

(a + b)³ = a³ + b³ + 3ab(a + b)

Tips

  • Combine like terms.
  • Apply identities wherever possible.
  • Simplify brackets using BODMAS rule.
  • Substitute values if variables are given.

Example

Simplify: (x + 2)² − (x − 3)²

⇒ [x² + 4x + 4] − [x² − 6x + 9]

⇒ 4x + 6x + 4 − 9 = 10x − 5

2. Probability (Basic)

Definition

Probability measures the likelihood of an event happening.

Probability (P) = Favourable Outcomes / Total Possible Outcomes

Important Points

Range

0 ≤ P(E) ≤ 1

P(E) = 0: Impossible event

P(E) = 1: Certain event

Complementary Events

P(E) + P(not E) = 1

Equally Likely Events

All outcomes have equal chances.

Example: Tossing a fair coin → {H, T}

Common Examples

Coin

2 outcomes

P(Head) = 1/2

Dice

6 outcomes

P(odd number) = 3/6 = 1/2

Cards (52 cards)

P(ace) = 4/52 = 1/13

P(king or queen) = 8/52 = 2/13

Example

Find the probability of drawing a red card from a deck of 52 cards.

Red cards = 26

P = 26/52 = 1/2

3. Permutation & Combination (Basic)

Permutation

Arrangement of objects in a particular order.

nPr = n! / (n - r)!

Example

How many ways can 3 students be seated in 3 chairs?

3P3 = 3! = 6 ways

Combination

Selection of objects without considering order.

nCr = n! / [r! × (n - r)!]

Example

How many ways can 2 students be chosen from 4?

4C2 = 4! / (2! × 2!) = 6

Key Differences

Concept Order Matters? Formula Example
Permutation Yes nPr = n! / (n - r)! Arranging 3 people in a line
Combination No nCr = n! / [r! × (n - r)!] Selecting 3 players for a team

Exam Tips for SSC MTS

  • Practice simplification of algebraic expressions regularly.
  • Revise factorial and basic probability questions from previous years.
  • Understand when order matters (Permutation) and when it doesn't (Combination).
  • Focus on simple real-life based problems (coins, dice, cards).
