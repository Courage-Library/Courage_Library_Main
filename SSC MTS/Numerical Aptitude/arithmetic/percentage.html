Percentage

Essential Concepts for Delhi Police & SSC Exams

1. Introduction

The Percentage chapter is one of the most important topics in SSC MTS, CHSL, and CGL exams. Almost every arithmetic topic — like Profit & Loss, Simple Interest, Time & Work, and Data Interpretation — uses the concept of percentage.

Let's learn all the key concepts, formulae, and tricks you need for full marks.

2. What is a Percentage?

The word 'percent' means 'per hundred'. It represents a number as a fraction of 100.

Percentage = (Value / Total Value) × 100

Example:

If a student scores 45 marks out of 60, then

Percentage = (45/60) × 100 = 75%

So, the student got 75% marks.

4. Basic Formulae of Percentage

(a) To find percentage of a number:

Required Value = (Given % / 100) × Total Value

Example: Find 20% of 250

→ (20/100) × 250 = 50

(b) To find what percent one number is of another:

Percentage = (Part / Whole) × 100

Example: What percent of 50 is 20?

→ (20/50) × 100 = 40%

(c) Increase or Decrease in Quantity:

If a number increases by x%:

New value = Original value × (1 + x/100)

If a number decreases by x%:

New value = Original value × (1 - x/100)

Example: If a salary of ₹10,000 is increased by 20%

→ New salary = 10,000 × (1 + 0.2) = ₹12,000

(d) To find % Increase or Decrease between two values:

% Change = (Difference / Original Value) × 100

Example: Price increases from ₹200 to ₹250

→ Change = 50

→ % Change = (50/200) × 100 = 25% increase

5. Successive Percentage Change

When a quantity changes two or more times by different percentages (increase or decrease), we use the Successive Percentage Formula:

Net Change = x + y + (xy)/100

Where x and y are the percentage changes. Use '+' for increase and '−' for decrease.

Example 1: Both Increases

A price is increased by 10% and then again by 20%. Find the net % increase.

→ Net % = 10 + 20 + (10 × 20)/100

→ 30 + 2 = 32% increase

Example 2: One Increase, One Decrease

Price increases by 20% and then decreases by 10%.

→ Net % = 20 – 10 – (20 × 10)/100

→ 10 – 2 = 8% increase

Example 3: Both Decreases

Value decreases by 10% and 20%.

→ Net % = –10 – 20 + (–10 × –20)/100

→ –30 + 2 = –28% decrease

6. Shortcut for Successive Change (Multiplicative Formula)

Final Value = Initial Value × (1 + x/100) × (1 + y/100)

Example:

A salary of ₹2000 is increased by 10% and then 20%.

→ Final = 2000 × 1.10 × 1.20 = ₹2640

Net increase = 640 → (640/2000) × 100 = 32%

Verified

7. Important Concept: Reversal of Percentage Change

When a value increases by x% and then decreases to reach the original value, the decrease % required is not x%, but slightly more.

Decrease % = (x × 100) / (100 + x)

Example:

If price increases by 25%, what % decrease will bring it back to the original?

→ (25 × 100) / (100 + 25) = 2500 / 125 = 20%

So, 20% decrease is required to return to original price.

8. Common Type of SSC MTS Questions

1. Find % Value:

What is 25% of 480?

→ (25/100) × 480 = 120

2. Find Base Value:

40% of what number is 120?

→ 120 = (40/100) × x

→ x = (120 × 100)/40 = 300

3. Find Increase/Decrease:

Value increases from 400 to 460

→ (460−400)/400 × 100 = 15% increase

4. Successive Change:

A number increases by 10% and 20% successively. Find net change.

→ 10 + 20 + (10×20)/100 = 32%

5. Reverse Change:

If price increases by 20%, what % decrease returns to old price?

→ (20×100)/(120) = 16.67%

9. Practice Questions

1. What is 12.5% of 240?

2. A number is increased by 10% and then decreased by 20%. Find the net % change.

3. If 45 is 15% of a number, find the number.

4. If a number is first decreased by 10% and then increased by 10%, what is the net % change?

5. The price of a shirt increased by 25% and then decreased by 20%. Find the final change.

10. Key Takeaways

Percentage = (Value / Total) × 100

Increase/Decrease Formula: New = Old × (1 ± x/100)

Successive Change Formula: x + y + (xy/100)

Reverse Change Formula: (x×100)/(100 + x)
