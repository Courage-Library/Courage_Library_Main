Earth's Structure – Crust, Mantle & Core

Complete Notes for SSC MTS, GD, CHSL - Physical Geography

Earth's Internal Structure

The Earth is composed of three main layers arranged like an onion. Understanding these layers is crucial for physical geography questions in SSC exams.

This is a very important & scoring topic. SSC MTS often asks about layers, composition, density, and thickness.

1. Crust

Outermost Layer

Thinnest, where we live

2. Mantle

Middle Layer

Thickest layer

3. Core

Innermost Layer

Hottest & densest

Memory Trick: "CMC = Crust → Mantle → Core"

1. Crust – The Outermost Layer

  • Thinnest layer of Earth
  • We live on the crust
  • Broken into tectonic plates
  • Lowest density among all layers

The crust is divided into two types based on location and composition.

Types of Crust

Continental Crust

  • Thickness: 30–70 km
  • Made of: Granite
  • Light in weight
  • Called SIAL

SIAL = Silica + Aluminium

Main elements in continental crust

Oceanic Crust

  • Thickness: 5–10 km
  • Made of: Basalt
  • Heavier than continental crust
  • Called SIMA

SIMA = Silica + Magnesium

Main elements in oceanic crust

Memory Trick

"Continental = Light SIAL, Oceanic = Heavy SIMA"

Key Points - Crust

Position: Outermost layer
Density: Lowest among all layers
Material: Granite (continental), Basalt (oceanic)
Special: Plate tectonics occurs here

2. Mantle – The Middle Layer

Features of Mantle

  • Thickness: 2,900 km (thickest layer)
  • Makes up 84% of Earth's volume
  • Temperature: 1000–3500°C
  • Rocks are semi-solid & molten

The mantle is responsible for tectonic plate movement and volcanic activity.

Upper Mantle Divisions

Asthenosphere

  • Soft, molten layer
  • Tectonic plates float on it
  • Enables plate movements

Lithosphere

  • Crust + Upper Mantle
  • Hard outer shell of Earth
  • Broken into tectonic plates

Composition

Mainly composed of:

Magnesium

Iron

MAFE = Magnesium + Ferrum (Iron)

Common abbreviation for mantle composition

Memory Trick & Key Points

"Mantle = MAFE, Soft Asthenosphere"

Composition: Magnesium, Iron
Character: Semi-molten
Special: Asthenosphere enables plate movement
Thickness: Thickest layer (2900 km)

3. Core – The Innermost Layer

The hottest, densest, and deepest layer of Earth.

Divided into two distinct parts with different physical states.

Despite extreme temperatures, the inner core remains solid due to immense pressure.

Divisions of the Core

Outer Core

  • State: Liquid
  • Composition: Iron + Nickel (NIFE)
  • Thickness: ~2200 km
  • Creates Earth's magnetic field

The flowing liquid metal generates Earth's protective magnetic field.

Inner Core

  • State: Solid (due to extreme pressure)
  • Composition: Iron + Nickel (NIFE)
  • Thickness: ~1300 km
  • Temperature: Up to 6000°C (equal to Sun's surface)

Hottest part of Earth, yet solid because pressure overcomes temperature effects.

Composition of Core

NIFE

NIckel + FErrum (Iron)

Both outer and inner core share this composition

Memory Trick & Comparison

"CORE = NIFE → NIckel + FErrum"

Feature Outer Core Inner Core
State Liquid Solid
Composition Iron-Nickel Iron-Nickel
Thickness ~2200 km ~1300 km
Special Magnetic Field Hottest Part

Summary Table (Super Important for SSC)

Layer State Composition Thickness Special Feature
Crust Solid SIAL (continental), SIMA (oceanic) 5–70 km Thinnest, tectonic plates
Mantle Semi-molten Magnesium + Iron (MAFE) 2900 km Asthenosphere, thickest layer
Outer Core Liquid Iron + Nickel (NIFE) ~2200 km Creates magnetic field
Inner Core Solid Iron + Nickel (NIFE) ~1300 km Hottest part (up to 6000°C)

SSC MTS Practice Questions

1. Which layer of Earth is the thinnest?

Crust (5-70 km thickness)

2. Which layer has SIMA composition?

Oceanic Crust (Silica + Magnesium)

3. What is the composition of Earth's core?

Iron + Nickel (NIFE)

4. Which layer creates Earth's magnetic field?

Outer Core (liquid iron-nickel layer)

5. What is the asthenosphere?

Soft, molten part of the upper mantle where tectonic plates float

6. Which is the thickest layer of Earth?

Mantle (2900 km thickness)

7. Which part of the core is solid?

Inner Core (solid due to extreme pressure)

8. What does SIAL stand for?

Silica + Aluminium (composition of continental crust)

Conclusion

Earth's internal structure is simple to remember if you focus on key associations: SIAL for continents, SIMA for oceans, MAFE for mantle, and NIFE for core. These layers work together to create the dynamic planet we live on.

Key Takeaways for SSC Exams

Crust types: Continental (SIAL) vs Oceanic (SIMA)
Mantle = thickest layer with asthenosphere
Core composition: NIFE (Iron + Nickel)
Outer core liquid → magnetic field
Inner core solid → hottest part
Lithosphere = Crust + Upper Mantle
