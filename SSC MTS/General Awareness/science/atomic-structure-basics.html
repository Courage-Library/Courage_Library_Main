Atomic Structure (Basics)

Chemistry - Complete Notes for SSC MTS, GD, CHSL

What is an Atom?

Atom = Smallest unit of matter that retains chemical properties of an element

Subatomic Particles

Particle Charge Mass Location
Proton (p⁺) +1 1 amu Nucleus
Neutron (n⁰) 0 1 amu Nucleus
Electron (e⁻) –1 1/1836 amu Orbits (shells)

Memory Trick: "Pro → Positive, Electron → Negative, Neutron → Neutral"

Atomic Number & Mass Number

Atomic Number (Z)

Z = Number of Protons

Since atom is neutral:

Protons = Electrons

Examples:

Hydrogen

Z = 1

Carbon

Z = 6

Oxygen

Z = 8

Mass Number (A)

A = Protons + Neutrons

A = Z + n

Example: Carbon-12

Protons:

6

Neutrons:

6

Mass Number = 12

Bohr's Model (VERY IMPORTANT)

Electrons revolve around nucleus in fixed shells/orbits

Shell Symbol Max Electrons Formula
1st K 2 2(1²) = 2
2nd L 8 2(2²) = 8
3rd M 18 2(3²) = 18
4th N 32 2(4²) = 32

Formula: Maximum electrons = 2n²

Where n = shell number (1, 2, 3, ...)

Electron Distribution (VERY IMPORTANT)

Oxygen (Z = 8)

2, 6

K shell:

2 electrons

L shell:

6 electrons

Sodium (Z = 11)

2, 8, 1

K shell:

2 electrons

L shell:

8 electrons

M shell:

1 electron

Chlorine (Z = 17)

2, 8, 7

K shell:

2 electrons

L shell:

8 electrons

M shell:

7 electrons

Valency (VERY IMPORTANT)

Valency = Combining capacity of an atom (based on outermost shell electrons)

Rules

Electrons ≤ 3:

Valency = number of electrons

Electrons ≥ 5:

Valency = 8 − electrons

Electrons = 4:

Valency = 4

Memory Trick: Valency = 8 – outer electrons (for non-metals)

Examples

Na (2,8,1)

Valency = 1

Mg (2,8,2)

Valency = 2

Cl (2,8,7)

Valency = 1 (8−7)

O (2,6)

Valency = 2

Isotopes & Isobars (100% Asked)

Isotopes

Definition:

Atoms of same element with same atomic number (Z) but different mass numbers (A)

Examples:

  • • Hydrogen: H¹, H², H³
  • • Carbon: C¹², C¹⁴

Uses:

  • • C¹⁴ → Carbon dating
  • • Co⁶⁰ → Cancer treatment

Memory Trick: Iso = Same (Protons same)

Isobars

Definition:

Atoms of different elements with same mass number (A) but different atomic numbers (Z)

Example:

Argon-40 & Calcium-40

Memory Trick: Bar = Same mass bar

Ions (Very Important)

Cations (Positive Ions)

Formation:

Atom loses electrons

Examples:

Na → Na⁺

Ca → Ca²⁺

Memory Trick: Loss = Positive

Anions (Negative Ions)

Formation:

Atom gains electrons

Examples:

Cl → Cl⁻

O → O²⁻

Memory Trick: Gain = Negative

Memory Tricks

PEN:

Protons (nucleus), Electrons (shells), Neutrons (nucleus)

Electron Filling:

2–8–8 Rule

Mass Number:

Protons + Neutrons

Isotope:

Same Z (Protons)

Isobar:

Same A (Mass)

Cation:

Loss → Positive

Anion:

Gain → Negative

Valency for non-metals:

8 – outer electrons

SSC MTS Practice Questions

1. What does atomic number represent?

Number of protons

2. Mass number equals?

Protons + Neutrons

3. Maximum electrons in L-shell?

8

4. Electron arrangement of sodium (Z = 11)?

2, 8, 1

5. What is valency of oxygen?

2

6. Isotopes are atoms with?

Same atomic number

7. Isobars are atoms with?

Same mass number

8. What is charge of electron?

–1

9. Ion formed when atom gains electron?

Anion

10. Which particle has no charge?

Neutron

Conclusion

Mastering Atomic Structure is crucial for SSC chemistry preparation. These concepts form the foundation of chemistry and consistently appear in competitive exams.

Key Takeaways

Proton: +1 charge, in nucleus
Electron: –1 charge, in shells
Neutron: 0 charge, in nucleus
Atomic number (Z): Number of protons
Mass number (A): Protons + Neutrons
Bohr's model: K(2), L(8), M(18), N(32)
Valency: Combining capacity (8–outer electrons)
Isotopes: Same Z, different A
Isobars: Same A, different Z
Cations: Positive ions (loss of electrons)
