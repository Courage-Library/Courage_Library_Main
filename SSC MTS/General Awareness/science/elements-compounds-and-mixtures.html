Elements, Compounds & Mixtures

Chemistry - Complete Notes for SSC MTS, GD, CHSL

Matter Classification

Matter is divided into Pure Substances and Impure Substances

Pure Substances

Elements:

Single type of atom

Compounds:

Chemically combined elements

Properties:

Fixed composition, definite properties

Impure Substances

Mixtures:

Physically combined substances

Composition:

Variable ratio

Properties:

Retains properties of components

Elements

An element is a pure substance made of only one type of atom

Metals

Properties:

  • • Shiny
  • • Good conductors
  • • Malleable & ductile

Examples:

Iron (Fe), Copper (Cu), Gold (Au)

Non-Metals

Properties:

  • • Dull
  • • Poor conductors
  • • Brittle

Examples:

Oxygen (O), Carbon (C), Nitrogen (N)

Metalloids

Properties:

Between metals & non-metals

Examples:

Silicon (Si), Boron (B), Germanium (Ge)

Memory Trick: B S A G Si Ge (common metalloids)

Compounds

A compound is formed by chemical combination of elements in fixed proportion

Characteristics

Formation:

By chemical reactions

Separation:

Cannot be separated by physical methods

Ratio:

Fixed proportion (H₂O always 2:1)

Properties:

Different from constituent elements

Common Examples

Water

H₂O (H + O)

Carbon Dioxide

CO₂ (C + O)

Common Salt

NaCl (Na + Cl)

Sugar

C₆H₁₂O₆

Ammonia

NH₃ (N + H)

Memory Trick: Compound = Chemically Combined

Mixtures

A mixture contains two or more substances physically combined

Homogeneous Mixture

Composition:

Uniform throughout

Examples:

  • • Salt water
  • • Sugar water
  • • Air
  • • Alloys

Memory Trick:

Homo = Same

Heterogeneous Mixture

Composition:

Uneven, visible components

Examples:

  • • Sand + water
  • • Oil + water
  • • Pizza
  • • Salad

Memory Trick:

Hetero = Different

Special Types of Mixtures

Alloys:

Homogeneous mixture of metals

Brass = Copper + Zinc

Bronze = Copper + Tin

Steel = Iron + Carbon

Solutions:

Homogeneous mixtures (solute + solvent)

Suspensions:

Large particles, settle down

Example: Muddy water

Colloids:

Intermediate particles

Example: Milk, fog

Key Differences (SSC Favorite)

Feature Element Compound Mixture
Composition One type of atom Fixed ratio Any ratio
Separation Cannot be separated Chemical methods Physical methods
Formation Natural Chemical reaction Physical mixing
Properties Same throughout New properties Properties retained
Examples O₂, Fe, Au H₂O, CO₂ Air, salt water

Separation Methods (Highly Important)

Filtration

Used for:

Solid + Liquid mixtures

Example: Sand + water

Evaporation

Used for:

Recover solid from solution

Example: Salt from seawater

Distillation

Used for:

Liquids with different boiling points

Example: Alcohol + water

Magnetic Separation

Used for:

Magnetic + non-magnetic mixtures

Example: Iron from sand

Chromatography

Used for:

Separating colors/components

Example: Ink components

Centrifugation

Used for:

Separating by density

Example: Cream from milk

Memory Tricks

Element:

Single atom type

Compound:

Chemically Combined

Mixture:

Mixed, Not Made

Homogeneous:

Homo = Same

Heterogeneous:

Hetero = Different

Alloy:

A lot of metals

Filtration:

Solid + Liquid

Distillation:

Different boiling points

SSC MTS Practice Questions

1. Which of these is a compound?

Water (H₂O)

2. Brass is a mixture of which metals?

Copper + Zinc

3. What type of mixture is air?

Homogeneous mixture

4. Which mixture can be separated by filtration?

Sand + Water

5. What type of mixture is milk?

Colloid

6. Which of these is an element?

Iron (Fe)

7. What is an alloy?

Homogeneous mixture of metals

8. Which method separates alcohol from water?

Distillation

9. How are mixtures separated?

Physical methods

10. What characterizes a compound?

Fixed ratio of elements

Conclusion

Mastering Elements, Compounds & Mixtures is essential for SSC chemistry preparation. These concepts form the foundation of chemistry and consistently appear in competitive exams.

Key Takeaways

Element: Single type of atom
Compound: Chemically combined elements
Mixture: Physically combined substances
Homogeneous: Uniform composition
Heterogeneous: Uneven composition
Alloy: Metal mixture (Brass, Bronze, Steel)
Filtration: Solid + Liquid separation
Distillation: Different boiling points
Evaporation: Salt from water
Magnetic Separation: Iron from sand
