Electricity & Magnetism

Physics - Complete Notes for SSC MTS, GD, CHSL

Electricity Basics

Electric Current (I)

I = Q / t

Flow of electric charge per second

Unit: Ampere (A)

Memory Trick: Current = Charge speed

Voltage (V)

Potential Difference

Difference in electric potential between two points

Unit: Volt (V)

Memory Trick: Voltage = Pressure pushing electrons

Resistance (R)

Opposition to current

Opposition to electric current flow

Unit: Ohm (Ω)

Conductors:

Low resistance (copper, aluminum)

Insulators:

High resistance (rubber, plastic)

Ohm's Law (VERY IMPORTANT)

V = I × R

Voltage = Current × Resistance

Understanding Ohm's Law

Voltage increases →

Current increases (if resistance constant)

Resistance increases →

Current decreases (if voltage constant)

Memory Trick

VIR

V = Voltage

I = Current

R = Resistance

Series & Parallel Circuits

Series Circuit

Path:

Single path for current

Current:

Same current flows through all components

Disadvantage:

If one bulb fuses → all turn off

Memory Trick:

Series = Single path

Parallel Circuit

Path:

Multiple paths for current

Voltage:

Same voltage across all branches

Advantage:

If one bulb fuses → others keep working

Memory Trick:

Parallel = Perfect for homes

Key Differences

Series:

Total resistance increases

Parallel:

Total resistance decreases

Heating Effect of Electric Current

H = I² × R × t

Joule's Law of Heating

Applications

Electric Iron:

Uses heating element

Electric Heater:

Converts electrical energy to heat

Incandescent Bulb:

Filament heats up to produce light

Geyser:

Heats water using heating element

Formula Explanation

H = Heat produced

Unit: Joule

I = Current

Unit: Ampere

R = Resistance

Unit: Ohm

t = Time

Unit: Second

Magnetic Effect of Electric Current

"Current produces magnetism" - Discovered by Hans Christian Oersted

Key Principle

When electric current flows through a conductor, it produces a magnetic field around it.

Electromagnet

Components:

  • • Soft iron core
  • • Coil of insulated wire
  • • Battery

Properties:

  • • Strong when current flows
  • • Loses magnetism when current stops
  • • Strength can be controlled

Memory Trick:

Electricity makes Magnet

Electric Bell (SSC FAVORITE)

Working Principle:

Electromagnetism

Working Steps:

  1. Switch ON → current flows
  2. Electromagnet attracts iron hammer
  3. Hammer hits bell → sound
  4. Circuit breaks → magnetism lost
  5. Hammer returns → cycle repeats

Magnetism Basics

Magnet Properties

Two Poles:

North (N)

South (S)

Magnetic Laws:

Like Poles

REPEL

Unlike Poles

ATTRACT

Magnetic Field Lines

Direction:

North → South outside the magnet

Characteristics:

  • • Never intersect each other
  • • Closer lines = stronger field
  • • Continuous closed loops

Memory Trick:

N→S outside the magnet

Earth's Magnetism

Earth behaves like a giant bar magnet

Key Facts

Magnetic South Pole ≈

Geographical North

Magnetic North Pole ≈

Geographical South

Reason:

Molten iron in Earth's outer core

Electromagnetic Induction

Discovered by:

Michael Faraday

Principle:

Current is induced when magnet moves near coil or coil moves near magnet

Applications:

  • • Electric generator
  • • Transformer

Memory Trick:

Movement → Current

AC vs DC Current

Direct Current (DC)

Flow:

Flows in one direction only

Examples:

  • • Batteries
  • • Power banks
  • • Solar cells

Voltage:

Constant voltage

Alternating Current (AC)

Flow:

Changes direction periodically

Examples:

  • • Household electricity
  • • Power grid

Frequency in India:

50 Hz

Memory Trick:

AC = Alternate / Changing

SSC MTS Practice Questions

1. What is the unit of electric current?

Ampere

2. What is Ohm's law equation?

V = I × R

3. What is the unit of resistance?

Ohm (Ω)

4. Which circuit is used in homes?

Parallel circuit

5. What produces magnetism when current flows?

Electric current

6. When does an electromagnet lose magnetism?

When current stops

7. What does current through a coil produce?

Magnetic field

8. What causes Earth's magnetic field?

Molten iron in outer core

9. What principle does an electric bell work on?

Electromagnetism

10. What is the frequency of AC in India?

50 Hz
