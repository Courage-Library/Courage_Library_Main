Alphabet Analogy

SSC MTS Exam Section

Concept

Alphabet analogy is based on letter relationship. We convert letters to positions for faster solving.

Position Values:

A=1, B=2, C=3 … Z=26

Example:

A : C :: E : ?

A=1 → C=3 ( +2 shift )

E=5 → +2 = G

Answer = G

Exam Tip: Always convert letters to numbers first to quickly identify the pattern.

Common Patterns used

Type Example Logic
Forward Shift A:C +2
Backward Shift F:D -2
Reverse Alphabet A:Z opposite side
Vowel / Consonant Relation A:O both vowel type
Position Math C:F 3 to 6 (×2)
Gap Based G:K +4 gap

Solved Examples (Short Answer)

D : G :: H : ?

D=4 → G=7 (+3)

H=8 → +3 = K

E : H :: P : ?

+3 pattern

P=16 → +3 = S

B : Y :: C : ?

Reverse alphabet

B→Y → 2→25

C→3 → opposite = X

Quick Recap

Alphabet analogy = shifting / reversing letter positions

Convert letters to numbers to quickly see patterns

Most used patterns: +2, +3, -2, reverse alphabet (A↔Z)

Reverse formula: Position of opposite letter = 27 - current position
