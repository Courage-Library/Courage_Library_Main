Mixed Series

SSC MTS Exam Section

Concept

Mixed Series contains combination of numbers + alphabets + sometimes symbols in one single sequence. Pattern can be in any one of them OR all elements may have their own separate pattern.

Example format:

A1 C2 E3 G4 …

Here alphabets follow +2 and numbers follow +1.

Exam Tip: Always analyze alphabet and number patterns separately first, then check if they interact.

Types of Mixed Series Patterns

Type Example Logic
Alphabet + Number separately A1, C2, E3, G4 Alphabet +2, Number +1
Alphabet forward / backward + Number reverse M5, N4, O3, P2 Alphabet +1, Number -1
Alternate Pattern A1, C1, A2, C2 Alphabet changes every two terms
Mixed skip logic B2, E3, H4, K5 Alphabet +3 and Number +1
Group wise pattern AB2, CD4, EF6 Pair +2, Number +2

Solved Examples

A1, C2, E3, G4, ?

→ Alphabet +2, Number +1

Ans: I5

K5, J6, I7, H8, ?

→ Alphabet -1, Number +1

Ans: G9

B3, D6, G10, K15, ?

→ +2 / +3 / +4 / +5 numbers increasing

Ans: P21

M2, P3, S4, V5, ?

→ +3 alphabet, +1 number

Ans: Y6

Quick Recap

Alphabet logic and number logic mostly run independently

Always check difference separately

Common increments: +2 alphabet +1 number / -1 alphabet +1 number

MTS keeps mixed series mostly simple arithmetic + alphabet jumps
