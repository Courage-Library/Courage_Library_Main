Figure Series

Visual Pattern Recognition - SSC MTS Exam Section

Concept

Figure series questions show a sequence of shapes/images, and you must identify the next figure based on pattern (rotation / addition / removal / shading / movement / repetition frequency). These are purely visual logic patterns, no language.

Exam Tip: Focus on one element at a time and track its changes systematically. Compare consecutive figures to spot the pattern.

Common Patterns in Figure Series

Pattern Type What to Observe
Rotation pattern figure rotates regularly (clockwise or anti-clockwise)
Element increase / decrease number of shapes increases or decreases
Shading pattern dark → light → dark etc.
Movement pattern particular object shifts position each step
Mirror flip alternate mirror reflection
Combination pattern rotation + shading / movement + addition

How to Solve Faster

Step 1

Track one element only → check how it's moving

Step 2

Check direction of rotation (CW or ACW)

Step 3

Look for +1 / -1 change in shape count

Step 4

Identify repeating cycle (like ABAC ABAC)

Step 5

In shading → dark / light mostly alternate pattern

Solved Examples (Short Logic)

A triangle rotates 90° clockwise in each step

→ next triangle will be again 90° rotated clockwise

Shapes: ● → ●● → ●●●

→ next is clearly 4 circles (+1 pattern)

Square shaded, next unshaded, next shaded…

→ next = unshaded (alternate pattern)

Practice Questions

1

In a series, a pentagon rotates 45° every step clockwise. What will happen next?

Options: a) rotate 45° again b) rotate 90° c) shading change d) mirror flip

Pattern continues consistently unless stated otherwise

View Answer

a) rotate 45° again

Rotation pattern continues consistently

2

A figure shows: 1 star → 2 stars → 3 stars → ?

Options: a) 5 b) 4 c) 6 d) 1

Look for +1 pattern in shape count

View Answer

b) 4

+1 pattern continues: 1→2→3→4

3

A shaded circle and unshaded circle alternate each step. After unshaded what comes next?

Options: a) unshaded b) shaded c) broken shape d) mirror shape

Alternate pattern: shaded → unshaded → shaded → unshaded

View Answer

b) shaded

Alternate pattern continues: shaded → unshaded → shaded

4

Arrow pointing up → right → down → left → ?

Options: a) up b) right c) left d) same

90° clockwise rotation pattern

View Answer

a) up

90° CW rotation: up→right→down→left→up (cycle completes)

5

In every next step, circle shifts one position to the right. Last was at 3rd position. Next will be at?

Options: a) 4th b) 1st c) 2nd d) same place

Movement pattern continues consistently

View Answer

a) 4th

Right shift continues: 3rd → 4th position

Quick Recap

6 main observation points: rotation, movement, shading, count, repetition cycle, mirror flip

Always compare only difference between two consecutive figures

Practice improves visual pattern spotting speed

Focus on consistency in pattern application
