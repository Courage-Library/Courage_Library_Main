Figure Classification

SSC MTS Exam Section

Concept

Identify which figure is different from others based on shape pattern, components, orientation, number of elements, shading style, rotation, line count etc.

Key Insight:

The odd figure is the one breaking a consistent hidden rule that applies to all other figures.

Exam Tip: Focus on identifying the common pattern first, then find which figure doesn't follow it.

How to Approach

Focus on property not on outer design

Check number of lines / identical shape count

Look for filled vs unfilled patterns

Check symmetry / location of shape

Identify consistent rule among majority

Find the breaker of the hidden rule

Common Patterns Used

Pattern Type Example Logic
Line count difference All have 3 lines but one has 4
Shading difference All are shaded right side except one
Shape orientation All arrowheads face right except one
Element presence/absence All have triangle + circle + square except one misses one
Symmetry rule break All symmetrical vertically except one
Pattern repetition count 2-2-2-2 but one is 3

Solved Examples

Example 1

Four shapes are shown – three have 2 circles inside a triangle, one has only 1 circle.

Odd: the one with 1 circle

Example 2

Four arrows – three point UP, one points LEFT

Odd: left pointing arrow

Example 3

Four figures – only one has filled dark square

Odd: the dark one

Quick Recap

Identify one consistent rule common to majority

Odd is the one breaking that rule

Check count of elements

Look for shading patterns

Check symmetry types

Verify rotation/orientation

Courage Tip: Master the art of pattern recognition by focusing on fundamental properties rather than surface appearances. Remember that the odd figure always breaks a consistent rule followed by all others. Practice systematically checking for count differences, symmetry, shading, orientation, and element composition to quickly identify the outlier.
