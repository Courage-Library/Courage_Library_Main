Creating & Editing Documents

Delhi Police Exam Section

1. Creating a Document

To create a new document in MS Word:

Using Blank Document

Open MS Word → Click Blank Document → Start typing.

Using Templates

MS Word provides pre-designed templates (letters, resumes, reports).

Navigate: File → New → Choose Template

Keyboard Shortcut

Ctrl + N → Create a new document quickly.

Exam Tip: Shortcut Ctrl + N is frequently asked in Delhi Police and SSC exams.

2. Opening & Saving Documents

Open Document

File → Open

or Ctrl + O

Save Document

File → Save

or Ctrl + S

Save As

Different formats

.docx, .pdf, .txt

Shortcut Reminder:

Save current document

Ctrl + S

Save As

Ctrl + Shift + S

3. Editing Documents

Basic Editing Tools

Tool Purpose Shortcut
Cut Remove selected text & place in clipboard Ctrl + X
Copy Copy selected text to clipboard Ctrl + C
Paste Insert copied/cut text Ctrl + V
Undo Revert last action Ctrl + Z
Redo Reverse undo Ctrl + Y
Select All Select entire document Ctrl + A

Advanced Editing Tools

Find & Replace → Locate specific words/phrases

Ctrl + F / Ctrl + H

Spell Check & Grammar → Review spelling

F7 or Review → Spelling & Grammar

Thesaurus → Find synonyms

Shift + F7

Track Changes → Review edits and collaborate

Review tab

5. Quick Recap

Create Document: Ctrl + N or Templates

Open Document: Ctrl + O

Save Document: Ctrl + S, Save As for different formats

Editing Shortcuts: Ctrl+C, Ctrl+X, Ctrl+V, Ctrl+Z, Ctrl+Y, Ctrl+A

Find & Replace: Ctrl + F / Ctrl + H

Spell Check: F7 | Track Changes: Review tab

You've completed Document Creation & Editing Concepts!

Courage Tip: Master the essential keyboard shortcuts for document operations - they are frequently tested in exams. Remember that Ctrl+N creates new documents, Ctrl+O opens existing ones, and Ctrl+S saves your work. Practice using Find & Replace and Track Changes for efficient document editing.
