Coding - Decoding

SSC MTS Exam Section

Concept

Coding–decoding questions give words/numbers and their coded forms. You must find the rule (letter shift, position change, reverse, substitution, arithmetic on digits) and apply it to decode or encode the required word/number.

Important:

Common in SSC MTS: simple, repeatable rules — don't overcomplicate.

Exam Tip: Always look for consistent patterns - SSC MTS uses clean, repeatable rules that are easy to identify.

Common Types of Coding–Decoding

Type Description Example
Letter Coding Alphabet based shifts A→C (+2), A→Z (mirror)
Number Coding Digit based operations Add/subtract constant, reverse digits
Mixed Coding Letters + numbers combined Alternate rules for odd/even positions
Word-level Coding Whole word transformations Synonyms/antonyms, anagrams

Pattern Recognition Tips (Fast)

Step 1

Check if every letter moves by same amount (uniform shift)

Step 2

Look for reverse/mirror: A→Z, B→Y etc.

Step 3

For digits: see if sum of digits, reverse or +k used

Step 4

Check positions: convert letters to numbers (A=1…)

Step 5

Test odd/even position rules separately

Step 6

Compare corresponding letters/digits pairwise

View Answer

Options: A) 012 B) 912 C) 012 D) 012

Pattern: Each digit +3 (mod 10)

View Answer

Options: A) 371 B) 371 C) 371 D) 371

Pattern: Digits reversed

View Answer

A) 371

5

Options: A) BEE B) BDE C) BEE D) BDE

Pattern: Each letter +1

View Answer

A) BEE

Quick Recap

Find consistent rule: shift / reverse / rotate / digit-op

Convert letters to positions if arithmetic suspected

Check odd/even position rules separately

MTS uses clean rules — expect +1/+2, reverse, rotate
