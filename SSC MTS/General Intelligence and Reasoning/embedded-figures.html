Embeddded Figures

SSC MTS Exam Section

Concept

In this type, a small given shape/figure is hidden inside a bigger complex figure. You need to identify in which option the exact same shape is present (same orientation / same size / same angle pattern).

Important Rule:

Only exact match qualifies — rotation may or may not be there (depends on question instruction).

Exam Tip: Look for exact structural match - same size, same angles, same orientation unless specified otherwise.

How to Solve Fast

Tip Explanation
Focus on corners & angles Easier than seeing full shape
Trace unique lines first Sharp bends / unique diagonal lines
Ignore shading / color Unless given specifically
Partial scanning Don't see the whole figure at once — scan part by part
Don't rotate mentally unless question allows SSC MTS usually SAME orientation

Solved Examples (Short)

Example 1

If question shows a zig-zag shape, find exact zig-zag inside big shape → ignore extra lines around.

Example 2

If shape has 3 straight edges + 1 diagonal, search same structure inside complex figure.

Example 3

If small figure has a unique triangle turn → find triangle exactly matching angle.

Quick Recap

Match exact line structure

Angles more important than shading

Ignore decorative lines

Start scanning from rare / unique edge pattern
