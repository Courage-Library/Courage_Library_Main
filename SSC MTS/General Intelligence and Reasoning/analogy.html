Analogy

SSC MTS Exam Section

What is Analogy?

Analogy means relationship based matching. You are given a pair of words / numbers / alphabets. Your task is to identify the relation between them and find another pair with the same relation.

Example:

Sun : Day :: Moon : ?

Relation → Sun gives Day → Moon gives Night

So answer = Night

Exam Tip: Always identify the relationship in the first pair first, then apply the same logic to find the missing term.

Types of Analogy

Word Analogy

Meaning / Function based

Number Analogy

Mathematical relationship

Alphabet Analogy

Position / Order based

1) Word Analogy

Relationship based on meaning / function / quality.

Example Relation Answer
Doctor : Hospital Worker - Workplace Teacher : School
King : Kingdom Ruler - Territory President : Country

Common relationships used:

Worker – Workplace

Tool – Usage

Product – Source

Country – Capital

Synonym / Antonym

2) Number Analogy

Check mathematical relationship.

Example Relation Answer
8 : 64 8² = 64 9 : 81

Common patterns used:

Square / Cube

Prime numbers

Addition / Multiplication

Difference based

3) Alphabet Analogy

Check alphabetical order / positions.

Example Relation Answer
A : C +2 shift E : G

Common patterns used:

+n / -n alphabet shifting

Reverse alphabet

Vowel / Consonant relation

Position based (A=1...Z=26)

Quicke Recognition Trick

Type Shortcut Formula
Word Identify meaning relation first
Number Always check square → cube → × pattern first
Alphabet Convert letters to numbers (A=1..Z=26)

Sample Questions

Question 1

Pen : Write :: Knife : ?

Question 2

5 : 125 :: 7 : ?

Question 3

C : F :: H : ?

Answers after thinking ↓

1. Cut

2. 343

3. K

Quick Recap

Analogy = finding similar relationship

Word analogy → meaning based

Number analogy → math logic (square / cube mostly)

Alphabet analogy → letter shifting / position numbers

Identify relation in first pair → apply same in second pair
