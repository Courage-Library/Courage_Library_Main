Classification / Odd One Out

SSC MTS Exam Section

Concept

In Classification / Odd One Out questions we identify which item doesn't belong to the same group as others.

Logic can be based on:

Category difference

Function difference

Meaning difference

Property difference

Usage difference

Example:

Apple, Banana, Mango, Potato

Potato is odd → because potato is a vegetable, others are fruits.

Exam Tip: Find the item that does NOT share the same common property with the majority of items.

Common Logic Patterns Used

Pattern Type Example Logic
Same Category Objects Animals / Fruits / Country etc.
Same Purpose / Function Devices used for communication etc.
Same Meaning Type Synonyms or Antonyms
Same Property Even numbers / Prime numbers etc.
Same Material / Source Items made of same basic thing

Solved Examples (Short Answer)

Doctor, Lawyer, Engineer, Bus

Odd one → Bus (others are professions)

Apple, Banana, Orange, Car

Odd one → Car (others are fruits)

Iron, Copper, Plastic, Gold

Odd one → Plastic (others are metals)

2, 4, 6, 9

Odd one → 9 (others are even numbers)

Look for meaning based grouping
