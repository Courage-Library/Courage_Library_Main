What is a Computer?

Delhi Police Exam

1. What is a Computer?

A computer is an electronic device that accepts data, processes it, stores it, and produces meaningful information as output.

Input → Process → Output → Storage

Programmable Electronic Device

Works on Given Instructions

Produces Meaningful Information

Simple Definition: A computer is a programmable electronic device that takes input, processes it according to given instructions, and provides output.

2. Characteristics of a Computer

Key Features

Characteristic Explanation Example
Speed Performs millions of calculations per second Solving complex equations instantly
Accuracy Delivers correct results when input is correct Banking transactions
Storage Can store vast data for future use Files, videos, documents
Versatility Can perform different types of tasks Watching movies, programming, designing
Diligence Doesn't get tired or bored Works 24×7 without breaks
Automation Works automatically once instructions are given Automatic billing, ATMs
Connectivity Connects to other devices and networks Internet, Wi-Fi, Bluetooth

3. Basic Components of a Computer

Five Main Components

1. Input Unit

Devices used to enter data into the computer

Example: Keyboard, Mouse, Scanner, Microphone

2. Processing Unit (CPU)

Known as the brain of the computer, it processes data and controls other parts

ALU (Arithmetic Logic Unit):

Performs mathematical and logical operations

CU (Control Unit):

Directs all operations inside the computer

Registers:

Small memory areas for temporary storage

3. Memory/Storage Unit

Stores data temporarily or permanently

Primary Memory:

RAM, ROM

Secondary Memory:

Hard Disk, Pendrive, CD/DVD

4. Output Unit

Displays or produces results after processing

Example: Monitor, Printer, Speaker

5. Communication Devices

Used to connect computers to networks or other systems

Example: Modem, Router, Bluetooth, Wi-Fi

4. Computer Working Cycle (IPO Cycle)

Input → Process → Output → Storage

Input

Data entered by user

Process

CPU processes data

Output

Result is displayed

Storage

Information saved for future

Example: MS Word

Typing text → (Input)

Editing text → (Process)

Displaying text → (Output)

Saving the file → (Storage)

5. Common Delhi Police Exam Facts

CPU is called the brain of the computer

ALU performs mathematical and logical operations

Device to convert data into readable form → Output device

RAM is volatile memory (data lost when power off)

Full form of ROM is Read Only Memory

Computer follows IPO Cycle

Concept Summary
Definition A computer is an electronic device that processes data into meaningful information
Key Features Speed, Accuracy, Storage, Versatility, Diligence
Main Units Input, Process, Storage, Output, Communication
Cycle Input → Process → Output → Storage
