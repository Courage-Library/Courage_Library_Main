Peripheral Devices

Delhi Police Exam

1. What Are Peripheral Devices?

Peripheral devices are external hardware components that are connected to the computer to either input data, output results, or store information.

External Hardware Components

Enhance Computer Functionality

Not Part of Core System

Connected Externally

Simply put: Peripherals = Devices that work with the computer from outside.

2. Types of Peripheral Devices

I

Input Devices

Used to enter data or commands into the computer.

Examples:

Keyboard

Mouse

Scanner

Microphone

Joystick

Webcam

Delhi Police PYQ 2020:

Which of the following is an input device?

Options: (a) Monitor (b) Printer (c) Keyboard (d) Speaker

Answer: (c) Keyboard

O

Output Devices

Used to display, print, or transmit information after processing.

Examples:

Monitor

Printer

Speaker

Projector

Plotter

Delhi Police PYQ 2021:

Which of these is an output device?

Options: (a) Mouse (b) Keyboard (c) Speaker (d) Scanner

Answer: (c) Speaker

S

Storage Devices

Used to store data permanently or temporarily.

Examples:

Hard Disk

Pen Drive

CD/DVD

Memory Card

SSD

Delhi Police PYQ 2022:

Pen drive is a type of:

Options: (a) Input device (b) Output device (c) Storage device (d) None

Answer: (c) Storage device

3. Internal vs External Peripherals

Type Examples Connection Type
Internal Peripherals Hard Disk, CD Drive, RAM Inside CPU cabinet
External Peripherals Keyboard, Mouse, Printer, Pen Drive Connected via USB or cable

4. Importance of Peripheral Devices

Expand computer functionality

Improve user interaction

Store and share data easily

Enable multimedia (audio/video input-output)

Allow printing, scanning, and digital communication

6. Quick Recap

Type Function Examples
Input Feed data Keyboard, Mouse, Scanner
Output Display results Monitor, Printer, Speaker
Storage Save data Hard Disk, Pen Drive, SSD

7. Memory Tip for Exam

"I-O-S" Formula

I – Input, O – Output, S – Storage

Every peripheral belongs to one of these three categories
