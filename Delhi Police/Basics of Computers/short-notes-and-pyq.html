Short Notes Summary - All Topics Covered

1. What is a Computer?

  • A machine that processes data into information
  • Characteristics: Speed, Accuracy, Automation, Storage, Connectivity
  • Components: Input, Processing (CPU), Output, Storage

2. Hardware vs Software

  • Hardware: Physical components (CPU, RAM, Keyboard, Monitor)
  • Software: Programs that run on hardware (System Software / Application Software)

3. Input, Processing, Output, Storage (IPO) Cycle

  • Input: Devices to enter data (Keyboard, Mouse)
  • Processing: CPU executes instructions
  • Output: Devices to display results (Monitor, Printer)
  • Storage: Save data (HDD, SSD, Pen Drive)

4. Types of Computers

  • Micro: Personal computers
  • Mini: Small business computers
  • Mainframe: Large-scale, high-performance computing
  • Supercomputer: Extremely high-speed computation

5. Generation of Computers

  • 1st Gen: Vacuum tubes
  • 2nd Gen: Transistors
  • 3rd Gen: Integrated Circuits (ICs)
  • 4th Gen: Microprocessors
  • 5th Gen: Artificial Intelligence / Advanced computing

6. Common Computer Terminology

  • Bit = 0 or 1
  • Byte = 8 bits
  • KB = 1024 Bytes
  • MB = 1024 KB
  • GB = 1024 MB
  • TB = 1024 GB

7. Peripheral Devices

  • Input Devices: Keyboard, Mouse, Scanner
  • Output Devices: Monitor, Printer, Speaker
  • Storage Devices: HDD, SSD, Pen Drive
  • Some devices are both Input/Output (e.g., Modem)
