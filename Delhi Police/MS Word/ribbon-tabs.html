Ribbon Tabs: Home, Insert, Layout, Review

Delhi Police Exam

1. What is a Ribbon in MS Word?

The Ribbon is the main toolbar in MS Word that organizes all commands into Tabs and Groups. Each tab contains related tools for performing specific tasks.

Main Toolbar in MS Word

Organized into Tabs & Groups

Located Below Title Bar

Quick Access to All Tools

Located below the Title Bar, the Ribbon helps you quickly access formatting, insertion, layout, and review tools.

2. Main Ribbon Tabs and Their Use

H

Home Tab

Used for basic text formatting and editing.

Group Important Commands
Clipboard Cut, Copy, Paste, Format Painter
Font Font style, size, color, Bold, Italic, Underline
Paragraph Alignment, Bullets, Line Spacing, Indent
Styles Apply pre-defined text styles
Editing Find, Replace, Select

Key Shortcut Actions:

Bold

Ctrl + B

Italic

Ctrl + I

Underline

Ctrl + U

Find

Ctrl + F

I

Insert Tab

Used for inserting objects and elements into a document.

Group Important Commands
Pages Cover Page, Blank Page, Page Break
Tables Insert and design tables
Illustrations Pictures, Shapes, Icons, SmartArt, Charts
Links Hyperlink, Bookmark, Cross-reference
Header & Footer Insert Header, Footer, Page Number
Text WordArt, Text Box, Signature Line
Symbols Insert Symbol, Equation

Useful Tip: Insert Tab is often used for adding visual or extra elements (tables, images, etc.).

L

Layout Tab

Used for page setup, margins, orientation, and spacing.

Group Important Commands
Page Setup Margins, Orientation (Portrait/Landscape), Size, Columns
Paragraph Indent and Spacing
Arrange Position, Wrap Text, Align, Rotate, Group

Useful Tip: Use Layout tab before printing — it controls the structure and appearance of your page.

R

Review Tab

Used for proofreading, tracking changes, and translation.

Group Important Commands
Proofing Spelling & Grammar, Thesaurus
Comments Add, Edit, Delete Comments
Tracking Track Changes, Show Markup
Changes Accept/Reject edits
Language Translate, Set Proofing Language
Protect Restrict Editing

Useful Tip: Review Tab is mainly used in editing and collaborative work.

5. Quick Recap

Ribbon Tabs = Home, Insert, Layout, Review

Home Tab → Font, Paragraph, Editing

Insert Tab → Tables, Pictures, Header/Footer

Layout Tab → Margins, Orientation, Spacing

Review Tab → Spelling, Track Changes, Comments
