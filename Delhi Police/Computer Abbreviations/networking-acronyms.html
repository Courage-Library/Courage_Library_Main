Networking Acronyms

Delhi Police Exam

Introduction to Networking Acronyms

The computer networking section often carries 2-3 questions in the Delhi Police Computer Exam.

Most of them are short acronym-based, focusing on basic internet and communication protocols.

Exam Insight: Focus on understanding what each acronym stands for and its basic function - this is enough to score marks in networking questions.

Essential Networking Acronyms

Acronym Full Form Meaning / Function
LAN Local Area Network Connects computers in a small area like office or building
WAN Wide Area Network Covers large geographical areas (like Internet)
MAN Metropolitan Area Network Covers a city or large campus
PAN Personal Area Network Small network around a single person (like Bluetooth)
IP Internet Protocol Provides unique addresses to devices on a network
DNS Domain Name System Converts website names (like google.com) to IP addresses
HTTP Hyper Text Transfer Protocol Transfers web pages from server to browser
HTTPS Hyper Text Transfer Protocol Secure Secure version of HTTP using SSL encryption
FTP File Transfer Protocol Used for transferring files between computers
SMTP Simple Mail Transfer Protocol Used to send emails
POP3 Post Office Protocol v3 Used to receive emails
IMAP Internet Message Access Protocol Accesses emails from mail servers
URL Uniform Resource Locator Address of a web page
ISP Internet Service Provider Provides internet connectivity (like Jio, Airtel)
MAC Media Access Control Unique ID assigned to every network device

Practice Questions (Delhi Police PYQ-style)

1

What is the full form of LAN?

Options: (a) Local Access Network (b) Local Area Network (c) Large Access Network (d) Linked Area Node

View Answer

Local Area Network

2

Which of the following is a wide area network?

Options: (a) Bluetooth (b) Internet (c) Wi-Fi router (d) Office Intranet

View Answer

Internet

3

The full form of IP is:

Options: (a) Internet Process (b) Internet Protocol (c) Interconnected Port (d) Internal Program

View Answer

Internet Protocol

4

DNS is used to:

Options: (a) Encrypt files (b) Assign IP addresses (c) Convert domain names to IP addresses (d) Manage server storage

View Answer

Convert domain names to IP addresses

5

What is the full form of HTTP?

Options: (a) High Text Transfer Protocol (b) Hyper Text Transfer Protocol (c) Hyper Text Test Program (d) Hyper Transfer Text Protocol

View Answer

Hyper Text Transfer Protocol

6

Which of these protocols is used for secure web browsing?

Options: (a) HTTP (b) HTTPS (c) FTP (d) POP3

View Answer

HTTPS

7

Which of the following is used for sending emails?

Options: (a) POP3 (b) FTP (c) SMTP (d) DNS

View Answer

SMTP

8

Which protocol helps in receiving emails?

Options: (a) SMTP (b) POP3 (c) HTTP (d) DNS

View Answer

POP3

9

What is the full form of URL?

Options: (a) Uniform Resource Locator (b) Unified Resource Link (c) Uniform Request Locator (d) Universal Retrieval Link

View Answer

Uniform Resource Locator

10

What is the role of ISP?

Options: (a) Connects users to the Internet (b) Sends emails (c) Hosts web pages (d) Stores passwords

View Answer

Connects users to the Internet

Memory Tricks & Exam Boosters

Network Types: Remember "Little Wise Men Prefer" - LAN → WAN → MAN → PAN

Email Protocols: SMTP = Send Mail, POP3 = Pickup Mail, IMAP = Internet Mail Access

Web Protocols: HTTP = Normal, HTTPS = Secure (S for Secure)

Address Systems: IP = Internet Address, MAC = Hardware Address, DNS = Name to Number

Exam Tip: Focus on understanding the basic function of each protocol - this helps eliminate wrong options quickly

Quick Recap

LAN → Local network (office/school)

WAN → Internet or large network

IP → Identity of each device

DNS → Translates names to numbers

HTTP/HTTPS → Transfers web pages (secure → HTTPS)

SMTP/POP3/IMAP → Email protocols

Additional Practice Questions

Q1. What is the main difference between HTTP and HTTPS?

View Answer

HTTPS is the secure version of HTTP that uses SSL encryption to protect data transmission

Q2. Why do we need both IP addresses and MAC addresses?

View Answer

IP addresses identify devices on a network (logical address), while MAC addresses identify the physical hardware (physical address)

Q3. What would happen if DNS servers stopped working?

View Answer

We wouldn't be able to access websites by their domain names, as browsers wouldn't be able to convert names to IP addresses

Q4. How does SMTP differ from POP3 in email communication?

View Answer

SMTP is used for sending emails from a client to a server or between servers, while POP3 is used for receiving emails from a server to a client

Q5. What is the relationship between LAN and WAN?

View Answer

LANs are smaller local networks that can connect to form larger WANs. The Internet is the largest WAN that connects millions of LANs worldwide
