Types of Printers

Delhi Police Exam

Introduction to Printer Types

Printers are common output devices asked in Delhi Police Computer Awareness exams.

Questions usually focus on type, function, and usage of different printers.

Exam Insight: Focus on understanding the working principle and specific use cases of each printer type - these are frequently tested in MCQs.

Printer Types Comparison

Printer Type Function / Features Usage / Notes
Laser Printer Uses laser beam to produce high-quality text and graphics Fast, ideal for offices; uses toner cartridges
Inkjet Printer Sprays tiny droplets of liquid ink onto paper Cost-effective, good for home printing and images
Dot Matrix Printer Prints by striking pins against an ink ribbon Noisy, low quality; used for multipart forms and invoices
