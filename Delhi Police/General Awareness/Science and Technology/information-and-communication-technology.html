Information & Communication Technology

Delhi Plice Exam

Introduction to ICT

Information & Communication Technology (ICT) forms the digital backbone of today's world — from smartphones to supercomputers, and from cloud data to cybersecurity.

In the Delhi Police exam, ICT questions test your understanding of computers, internet basics, networking, and digital safety.

Memory Trick: "Connect, Compute, Communicate — that's ICT."

Computers & IT Basics

Internet & Its Evolution

Concept Explanation
Internet A global network connecting millions of computers
WWW (World Wide Web) A collection of websites accessible via the Internet
Browser Software to access web pages (Chrome, Edge, Firefox)
Search Engine Helps find information (Google, Bing, DuckDuckGo)
Trick: "Internet connects; Web reflects."

Cloud Computing

Category Details
Definition Technology that stores data and runs applications over the Internet instead of a local computer
Examples Google Drive, iCloud, Microsoft Azure, Amazon AWS
Benefits Anywhere access, scalability, low cost
Trick: "Data on Cloud, Access Allowed."

Artificial Intelligence (AI)

Category Details
Definition Simulation of human intelligence in machines
Applications Chatbots, facial recognition, self-driving cars, language translation
Subfields Machine Learning, Deep Learning, Natural Language Processing (NLP)
Trick: "AI = Machines that Think."

Machine Learning (ML)

Category Details
Definition A branch of AI where machines learn from data without explicit programming
Examples Spam filters, product recommendations, speech recognition
Trick: "ML Learns from Experience."

Robotics

Category Details
Definition Science of designing and using robots for human-like or repetitive tasks
Applications Manufacturing, space missions, healthcare, defense
Example ISRO's "Vyommitra" robot (for Gaganyaan mission)
Trick: "Robotics = AI with Hands."

Cybersecurity

Category Details
Definition Protection of systems and data from cyberattacks
Common Threats Phishing, Hacking, Malware, Ransomware
Tools Firewalls, Antivirus, Encryption, Password protection
Trick: "Cybersecurity – Guard your Digital Identity."

Cybercrime Examples

Type Example
Phishing Fake emails to steal info
Hacking Unauthorized access to data
Identity Theft Stealing personal data
Cyberbullying Online harassment
Trick: "Phish steals, Hack reveals."

Firewalls & Antivirus

Tool Function
Firewall Blocks unauthorized network access (Acts like a digital security gate)
Antivirus Detects and removes malware (e.g., Norton, McAfee, Quick Heal)
Trick: "Firewall blocks, Antivirus knocks out viruses."

Networking Basics

Types of Networks

Type Full Form Range / Example
LAN Local Area Network Small area (office, school)
MAN Metropolitan Area Network City-level
WAN Wide Area Network Large area (Internet)
Trick: "LAN – Local, MAN – Metro, WAN – World."

IP Address

Category Details
Definition A unique number assigned to each device on a network
Versions IPv4 (e.g., 192.168.1.1), IPv6 (newer format)
Purpose Identifies devices for data transfer
Trick: "IP = Internet Passport."

DNS (Domain Name System)

Category Details
Function Converts domain names (like www.google.com) into IP addresses
Analogy Like a phonebook for the Internet
Trick: "DNS remembers names, not numbers."

VPN (Virtual Private Network)

Category Details
Definition A secure connection that hides your IP and encrypts data over public networks
Use Privacy protection, secure browsing, remote work
Trick: "VPN = Virtual Privacy Network."

Software & Internet Acronyms

Common Internet Terms

Term Full Form Meaning
HTML HyperText Markup Language Used to create web pages
HTTP HyperText Transfer Protocol Transfers web data
HTTPS Secure version of HTTP (encrypted) Secure data transfer
URL Uniform Resource Locator Web address (www.example.com)
IP Internet Protocol Unique device identifier
DNS Domain Name System Converts domain to IP
PDF Portable Document Format Universal document format
API Application Programming Interface Allows software to communicate
AI Artificial Intelligence Machine intelligence
ML Machine Learning Learning from data
Trick: "H-for-HTML/HTTP, U-for-URL, P-for-PDF, A-for-API."

Software Types

Type Example
System Software Windows, Linux, macOS
Application Software MS Word, Excel, Photoshop
Utility Software Antivirus, File Compression
Open Source Software LibreOffice, Linux
Trick: "System runs, Application works, Utility protects."

Practice Questions (Delhi Police, SSC & State Exams)

1

The full form of HTML is —

A) HyperText Making Language   B) HyperText Markup Language   C) HighText Marking Language   D) Hyper Transfer Markup Language

Category: Internet Terms

Show Answer

B) HyperText Markup Language

2

Which of the following is India's humanoid robot for space missions?

A) Vyommitra   B) Sophia   C) Mitra   D) Titan

Category: Robotics

Show Answer

A) Vyommitra

3

The main function of a Firewall is —

A) Virus scanning   B) Blocking unauthorized network access   C) Data backup   D) Compressing files

Category: Cybersecurity

Show Answer

B) Blocking unauthorized network access

4

The full form of VPN is —

A) Virtual Public Network   B) Virtual Private Network   C) Variable Protocol Network   D) Verified Private Network

Category: Networking

Show Answer

B) Virtual Private Network

5

Which of the following is an example of Cloud Computing?

A) Microsoft Word   B) Google Drive   C) Paint   D) Excel

Category: Cloud Computing

Show Answer

B) Google Drive

6

Which protocol is used for secure website communication?

A) HTTP   B) HTTPS   C) HTML   D) URL

Category: Internet Terms

Show Answer

B) HTTPS
