Shortcut Keys in Excel

Complete Guide for Delhi Police Computer Section

1. Why Shortcut Keys Matter

In exams like Delhi Police Constable or SSC CGL, questions from Excel shortcut keys are super common — and sometimes the easiest to score!

Keyboard shortcuts save time and help you work faster than the mouse.

Exam Insight: In government exams, 2-3 questions are directly asked on shortcuts — like "Which shortcut is used to insert a new worksheet in Excel?"

2. Basic Shortcut Keys

Action Shortcut Key Trick to Remember
Create a new workbook Ctrl + N N → New
Open existing workbook Ctrl + O O → Open
Save workbook Ctrl + S S → Save
Print workbook Ctrl + P P → Print
Close workbook Ctrl + W W → Window close

Pro Tip: Always save before printing — Ctrl + S → Ctrl + P (Safe Practice Combo)

3. Editing & Formatting Shortcuts

Action Shortcut Key Trick
Copy Ctrl + C C → Copy
Cut Ctrl + X X → Cut cross
Paste Ctrl + V V → Vase (Paste)
Undo Ctrl + Z Z → Zip back
Redo Ctrl + Y Y → Yess again!
Bold Ctrl + B B → Bold
Italic Ctrl + I I → Italic
Underline Ctrl + U U → Underline

4. Worksheet Navigation Shortcuts

Action Shortcut Trick
Move to next sheet Ctrl + Page Down Down → Next
Move to previous sheet Ctrl + Page Up Up → Previous
Insert new worksheet Shift + F11 F11 → Fresh sheet
Select entire sheet Ctrl + A A → All
Go to specific cell Ctrl + G G → Go to
Move to cell A1 Ctrl + Home Home → Beginning

Exam Trick: Many confuse Ctrl + N (new workbook) with Shift + F11 (new sheet) — Remember: N = New File, Shift + F11 = New Sheet inside file

5. Data & Formula Shortcuts

Action Shortcut Tip
Insert current date Ctrl + ; Semicolon = date separator
Insert current time Ctrl + Shift + ; Add Shift for time
AutoSum Alt + = Adds selected cells fast
Edit active cell F2 F2 → Fix content
Calculate all sheets F9 9 → Total
Display formulas Ctrl + ~ Tilde (~) → Show all formulas

7. Smart Tricks & Exam Boosters

Trick 1: Think alphabetically: C → Copy, V → Paste, X → Cut, S → Save, P → Print

Trick 2: "Shift" always means something bigger or additional → Shift + F11 adds a whole new sheet!

Trick 3: "Ctrl + Arrow" jumps to edges — perfect for long data lists

Trick 4: To quickly select data region → Ctrl + Shift + Arrow

Time Tip: For time + date stamp: Ctrl + ; (date) → Ctrl + Shift + ; (time) — just like day & night combo

10. Practice Questions

Q1. What is the difference between Ctrl+N and Shift+F11?

View Answer

Ctrl+N creates a new workbook (file), while Shift+F11 inserts a new worksheet (tab) within the current workbook

Q2. How would you quickly navigate to cell Z100 in a large worksheet?

View Answer

Use Ctrl+G (Go To) and type Z100, or use Ctrl+Arrow keys to navigate quickly

Q3. What happens when you press Ctrl+~ in Excel?

View Answer

It toggles between displaying cell values and displaying the actual formulas in cells

Q4. Why is it important to use keyboard shortcuts in Excel?

View Answer

Keyboard shortcuts save time, increase productivity, reduce reliance on mouse, and are frequently tested in competitive exams

Q5. How can you quickly select an entire column or row?

View Answer

Use Ctrl+Space to select entire column, and Shift+Space to select entire row
