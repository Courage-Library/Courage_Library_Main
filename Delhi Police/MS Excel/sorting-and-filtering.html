Sorting & Filtering in Excel

Data Organization Guide for Delhi Police / SSC Exams

Excel isn't just about typing data — it's about organizing it smartly! When your data looks messy, Excel's Sorting & Filtering features act like Delhi Police traffic control — keeping everything in order and easy to search!

What is Sorting?

Sorting means arranging data in a specific order — ascending (A→Z / 0→9) or descending (Z→A / 9→0).

Shortcut: Alt + D + S opens the Sort dialog box.

Example:

Name Marks
Raj 88
Aman 72
Neha 91

If you sort Marks in descending order → Neha → Raj → Aman

Types of Sorting

Type Example Description
Alphabetical Sorting A→Z or Z→A Used for text
Numerical Sorting 0→9 or 9→0 Used for numbers
Date Sorting Oldest→Newest or vice versa Used for date values
Custom Sort Sort by color, cell icon, or multiple columns For advanced data control

Trick: If you want to sort by two columns, like Department first and then Name → Use Custom Sort → Add Level.

What is Filtering?

Filtering means showing only the data you need and hiding the rest (without deleting it).

Shortcut: Ctrl + Shift + L = Add or remove filters instantly.

Example:

From a list of 500 records, you only want "Delhi" candidates → Apply a filter and see only those rows!

Types of Filters

Filter Type Example Purpose
AutoFilter Filter specific names, cities Most common
Number Filter >, <, Between For numeric data
Text Filter Begins with, Ends with, Contains For text
Date Filter Today, Last Month, Between For dates
Advanced Filter For complex conditions Custom filtering

Pro Tip: Use "Contains" when you're not sure of the full name — like searching "del" for "Delhi".

Smart Tricks & Shortcuts

Always include column headers before sorting — otherwise your data may jumble!

Use color-based sorting to prioritize highlighted data. → Sort → Custom Sort → "Sort On: Cell Color"

Want to remove filter quickly? Press Ctrl + Shift + L again.

Use Filter by Color when you've highlighted rows for review — saves time!

To clear all filters → Go to Data tab → "Clear Filter."
