Comprehensive Excel Assessment for Delhi Police / SSC Exams

Objective Questions (PYQs & Practice)

1

Which shortcut is used to create a new workbook?

Options: (a) Ctrl + O (b) Ctrl + N (c) Shift + F11 (d) Ctrl + S

Category: Shortcuts

Show Answer

Ctrl + N

2

What does the shortcut Alt + = do in Excel?

Options: (a) Copy (b) AutoSum (c) Paste (d) Filter

Category: Shortcuts

Show Answer

AutoSum

3

Which of the following is NOT an Excel data type?

Options: (a) Number (b) Text (c) Date (d) Paragraph

Show Answer

4

To apply conditional formatting, you go to which tab?

Options: (a) Home (b) Insert (c) Data (d) Review

Category: Formulas

Practical Task – Make a Marksheet

Task: Create an Excel marksheet for 5 students

Required Structure:

Student Name Roll No Maths English Science Total Percentage Grade

Instructions:

Fill random marks (0–100) for each subject
Use SUM formula to calculate Total
Use AVERAGE formula to calculate Percentage

Conditional Formatting Requirements:

Marks < 33 → Red fill

Marks ≥ 75 → Green fill

Pro Tips:

  • Use AutoSum for Total column
  • Use IF formula to assign Grades (e.g., =IF(F2>=75,"A",IF(F2>=50,"B","C")))
  • Use Color Scales to visualize highest and lowest marks
  • Apply Borders, Bold Headings, and Center Alignment
  • Save the file as Marksheet_Practice.xlsx
