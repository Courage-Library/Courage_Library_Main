Excel Interface & Cell Basics

Delhi Police Exam

1. What is Microsoft Excel?

Microsoft Excel is a spreadsheet application used to store, organize, calculate and analyze data in rows and columns. It's part of the MS Office suite and is widely used for tables, calculations, charts, and basic data analysis.

2. Excel Workbook vs Worksheet

üìÅ Workbook

The entire Excel file (one .xlsx file)

üìÑ Worksheet (Sheet)

A single tab inside the workbook (Sheet1, Sheet2...). You can have many worksheets in one workbook.

4. Cells ‚Äî The Building Blocks

Cell = intersection of a row and a column (example: A1)

Cell Address/Reference: Column letter + Row number (A1, B3, C10)

Editing cell: Double-click a cell, or select cell and press F2 to edit in-place

Selecting cells: Click single cell; click & drag to select range; Shift + Arrow to extend selection

Entering data: Type directly into cell or into the Formula Bar; press Enter to move down, Tab to move right

üìù Text

6. Cell References

Relative Reference (A1)

Changes when copied (A1 ‚Üí B1 if copied one column right)

Absolute Reference ($A$1)

Fixed when copied

Mixed Reference ($A1 or A$1)

One part fixed

Range Reference (A1:B5)

Block of cells

Exam Tip: Questions often ask which reference to use to keep a row/column fixed in formulas ‚Äî answer = $ absolute

7. Formula Bar & Basic Formulas

All formulas start with =

Examples: =SUM(A1:A10), =AVERAGE(B2:B12), =IF(C2>50,"Pass","Fail")

AutoSum button (Home tab / Formulas tab) quickly inserts =SUM(...)

Use F4 while editing a formula to toggle relative/absolute references
