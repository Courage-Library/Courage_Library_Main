Web Browsers and Their Functions

Delhi Police Exam - Computer Knowledge

1. Introduction

A Web Browser is a software application that allows users to access, view, and interact with websites on the Internet. It acts as a bridge between the user and the World Wide Web (WWW).

Simple Definition: A Web Browser is a program used to open and view web pages on the Internet.

2. What is a Web Browser?

Feature Description
Meaning A software that displays web pages and websites.
Purpose To access information from the World Wide Web.
Developed By Different tech companies (e.g., Google, Microsoft, Mozilla).
Main Function Converts website code (HTML, CSS, JS) into readable content.
Examples Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, Opera

üí° Tip: Without a web browser, you cannot open or view websites on your device.

3. Examples of Popular Web Browsers

Browser Developer Key Features
Google Chrome Google Fast, secure, supports extensions, most used worldwide
Mozilla Firefox Mozilla Foundation Open-source, privacy-focused
Microsoft Edge Microsoft Built on Chromium, integrates with Windows
Safari Apple Inc. Default browser for macOS and iOS
Opera Opera Software Built-in VPN, data-saving mode
Brave Brave Software Blocks ads and trackers automatically

üèÜ Exam Tip: Google Chrome is the most popular web browser globally and in India.

4. Main Functions of a Web Browser

Function Description
1. Access Websites Opens web pages using URLs (e.g., https://www.couragelibrary.in).
2. Display Content Shows text, images, videos, and graphics properly formatted.
3. Navigate Pages Allows moving between web pages via hyperlinks or back/forward buttons.
4. Download Files Enables saving files like PDFs, images, and videos.
5. Manage Tabs Lets users open multiple websites in one window.
6. Store History Keeps a record of visited websites.
7. Bookmark Pages Saves frequently used websites for quick access.
8. Ensure Security Uses HTTPS and warns about unsafe sites.

üîí Security Tip: Modern browsers use HTTPS to provide secure communication over the Internet.

5. Key Components of a Web Browser

Component Function
Address Bar (URL Bar) Where you type the website address.
Navigation Buttons Back, Forward, Refresh, and Home buttons.
Tabs Allow multiple websites to open in one window.
Bookmark Bar Stores quick links to favorite sites.
Search Box Used to search directly from the browser.
Settings/Tools Menu Customize appearance, privacy, and extensions.
Extensions/Add-ons Enhance browser functionality (like Grammarly or AdBlock).

üí° Example: In Google Chrome, the address bar is also a search bar, called the Omnibox.

6. Working of a Web Browser

1

User types a URL (e.g., www.couragelibrary.in)

2

Browser contacts the web server through the Internet

3

The server sends back the webpage data (HTML, CSS, JS)

4

The browser processes this data and renders it visually on the screen

üñ•Ô∏è Technical Term: The process of showing a web page on your screen is called Rendering.

7. Difference Between Web Browser and Search Engine

Basis Web Browser Search Engine
Meaning Software used to access and view websites. Tool used to find websites and information.
Function Displays web pages. Searches for web pages.
Examples Chrome, Firefox, Edge. Google, Bing, Yahoo.
Dependency Needs a search engine to search. Needs a browser to open.
Output Displays web content. Shows a list of websites.

‚ö†Ô∏è Exam Alert: Google Search = Search Engine  |  Google Chrome = Web Browser

8. Real-Life Analogy

Think of the Internet as a massive city:

Internet

City full of information

Search Engine

A map or guide that helps you find locations

Web Browser

Your vehicle that takes you to that location

In Short: Search Engine finds websites ‚Üí Browser opens and displays them.

Examples Chrome, Edge, Safari, Firefox
