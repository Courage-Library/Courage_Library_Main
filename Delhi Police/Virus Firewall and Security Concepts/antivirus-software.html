Antivirus Software

Delhi Police Exam

1. What is Antivirus Software?

Antivirus software is a program designed to detect, prevent, and remove malicious programs like viruses, worms, trojans, and ransomware.

Exam-Focused Definition: Antivirus protects your computer by scanning, detecting, and removing harmful software.

Key Functions:

Real-time scanning of files and programs

Automatic removal of threats

Periodic system scans

Updates to detect new viruses

2. How Antivirus Works

1. Scanning Files

Checks files for known virus patterns (signatures)

2. Heuristic Analysis

Detects unknown viruses by analyzing suspicious behavior

3. Quarantine

Isolates infected files to prevent further damage

4. Automatic Updates

Keeps virus definitions up-to-date to tackle new threats

3. Common Antivirus Software (Examples)

1

Which of the following is an antivirus software?

Options: (a) Kaspersky (b) Photoshop (c) Windows Media Player (d) MS Word

View Answer

Kaspersky

2

The main function of antivirus is to:

Options: (a) Edit documents (b) Remove malware (c) Design graphics (d) Manage files

View Answer

Remove malware

3

Which of the following antivirus is free for basic use?

Options: (a) Norton (b) Avast (c) McAfee (d) Bitdefender

View Answer

Avast

4

Heuristic analysis in antivirus software is used to:

Options: (a) Detect unknown viruses (b) Remove old files (c) Improve internet speed (d) Update the OS

View Answer

Detect unknown viruses

5

Quarantine feature in antivirus software:

Options: (a) Deletes all files permanently (b) Isolates infected files (c) Formats the hard drive (d) Updates software

View Answer

Isolates infected files

Short Answer Task

Question:

Explain in 50 words the primary purpose of antivirus software and give two examples commonly used in India.

Hint: Use keywords like detection, prevention, Kaspersky, Norton.

Sample Answer:

Antivirus software protects computers by detecting, preventing, and removing malware like viruses and trojans. It scans files, uses heuristic analysis, and quarantines threats. Popular examples include Kaspersky for real-time protection and Norton for malware removal, both widely used in India for cybersecurity.
