Internet Security

Delhi Police Exam

1. What is Internet Security?

Internet Security refers to the practices and tools used to protect computers, networks, and data while using the internet. It ensures that your personal information, devices, and online activities remain safe from cyber threats.

Exam-Focused Definition: Internet security protects against unauthorized access, malware, phishing, and other cyber attacks.

2. Importance of Strong Passwords

A strong password is the first line of defense for your online accounts.

Characteristics of a Strong Password:

At least 8–12 characters

Combination of uppercase, lowercase, numbers, and special symbols

Avoid common words or sequences (e.g., 123456, password)

Use unique passwords for different accounts

Trick to Remember: Long + Complex + Unique = Strong Password

3. Safe Browsing Practices

Check URLs: Ensure websites use HTTPS

Avoid suspicious links or pop-ups

Do not download files from untrusted sources

Use updated browsers and antivirus software

Clear cache and cookies regularly

Enable two-factor authentication wherever possible

Exam Trick: Safe browsing = "Think before you click!"

4. Delhi Police PYQ Style Questions

1

Which of the following is a characteristic of a strong password?

Options: (a) 123456 (b) Abc@2025! (c) password (d) admin

View Answer

Abc@2025!

2

HTTPS in a website URL indicates:

Options: (a) Unsafe connection (b) Encrypted and secure connection (c) Free browsing (d) Website under maintenance

View Answer

Encrypted and secure connection

3

Which practice is not recommended for safe browsing?

Options: (a) Clicking suspicious links (b) Using updated browsers (c) Avoiding public Wi-Fi for sensitive transactions (d) Enabling 2FA

View Answer

Clicking suspicious links

4

Two-factor authentication (2FA) is used to:

Options: (a) Speed up internet (b) Add extra security layer (c) Install software (d) Encrypt hard drive

View Answer

Add extra security layer

5

Which of the following is considered unsafe?

Options: (a) Downloading files from official sites (b) Using unknown email attachments (c) Regular antivirus updates (d) Enabling firewall

View Answer

Using unknown email attachments

Short Answer Task

Question:

Write 5 safe browsing tips in 50 words or less and explain why strong passwords are essential for online security.

Sample Answer:

Safe browsing tips: Use HTTPS websites, avoid suspicious links, don't download from untrusted sources, update software regularly, enable 2FA. Strong passwords are essential as they prevent unauthorized access, protect personal data, and serve as the first defense against cyber attacks and identity theft.

5. Quick Recap / Memory Tricks

Concept Trick to Remember
Strong Password "Long + Complex + Unique"
HTTPS "S = Secure"
Safe Browsing "Think before you click"
Two-Factor Authentication "Extra lock for accounts"

Exam Tip: In Delhi Police exams, MCQs usually ask password rules, safe browsing practices, HTTPS meaning, and 2FA purpose. Memorizing the table helps cover most questions.

Practical Assignment

Task: Security Self-Assessment

Conduct a personal internet security audit and implement security improvements.

1

Check if your important accounts have strong passwords

2

Enable two-factor authentication on email and social media accounts

3

Verify that your browser shows HTTPS on websites you visit

4

Update your antivirus software and browser to latest versions

5

Create a list of 5 security improvements you implemented

You've completed Internet Security!

Remember that strong passwords, safe browsing habits, and security features like HTTPS and 2FA work together to protect your online presence. Regular security audits and staying updated with best practices will help you stay protected both in exams and real-world scenarios.
