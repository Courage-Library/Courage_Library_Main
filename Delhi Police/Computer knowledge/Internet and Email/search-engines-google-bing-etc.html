Search Engines (Google, Bing, etc.)

Delhi Police Exam - Computer Knowledge

1. Introduction

A Search Engine is a software program that helps users find information on the Internet. It searches millions of web pages and shows results based on the words or phrases (called keywords) entered by the user.

Simple Definition: A Search Engine is a tool that locates information on the Internet using keywords and displays relevant results.

2. How Does a Search Engine Work?

A search engine performs three main tasks:

Step Function Description
1. Crawling Data Collection The search engine visits web pages using bots or spiders to collect information.
2. Indexing Data Storage The collected data is organized in a database (index) for quick access.
3. Searching & Ranking Result Display When a user searches, it shows the most relevant results based on ranking algorithms.

üí° Tip: The process of improving a website's visibility on search engines is called SEO (Search Engine Optimization).

4. Basic Components of a Search Engine

Component Description
Crawler (Spider) Program that scans and collects website data.
Indexer Organizes and stores data for quick retrieval.
Database Stores the indexed web pages.
Ranker/Algorithm Determines which pages appear first.
User Interface The search box and results page seen by users.

üí° Tip: Search engines use ranking algorithms to decide which result appears at the top.

7. Difference Between Browser and Search Engine

Basis Browser Search Engine
Meaning Software to access web pages Tool to find web pages
Examples Chrome, Firefox, Edge Google, Bing, Yahoo
Function Displays websites Helps locate websites
Dependency Needs a search engine for search Needs a browser to open
Storage Saves bookmarks, history Stores indexed web data
