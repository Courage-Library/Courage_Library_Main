Quick Revision Notes

Delhi Police Exam

This comprehensive revision guide covers all essential computer awareness topics for the Delhi Police Exam. Use these quick notes for last-minute revision and to reinforce key concepts that frequently appear in competitive exams.

1. Computer Basics

Key Definitions:

  • Definition: Electronic device that processes data to produce meaningful information
  • Characteristics: Speed, accuracy, storage, automation
  • Components: Input, Processing, Output, Storage (IPOS cycle)

Types of Computers:

  • Micro, Mini, Mainframe, Super

Computer Generations:

1st Gen: Vacuum tubes

2nd Gen: Transistors

3rd Gen: ICs

4th Gen: Microprocessors

5th Gen: AI-based computers

Exam Trick: Remember "V-T-I-M-A" = Vacuum → Transistor → IC → Microprocessor → AI.

2. Hardware & Software

Hardware:

Physical components (CPU, Monitor, Keyboard, Mouse)

Software:

System Software: OS, Utilities

Application Software: Word Processor, Excel, Browser

Peripheral Devices:

Input Devices:

Keyboard, Mouse, Scanner, MICR, OMR, OCR

Output Devices:

Monitor, Printer, Speaker, Projector, Plotter

3. MS Word & Excel Basics

MS Word:

Interface:

Ribbon tabs (Home, Insert, Layout, Review)

Elements:

Table, Image, Header/Footer, Page Number

Shortcut Keys:

Ctrl + C → Copy
Ctrl + V → Paste
Ctrl + B → Bold
Ctrl + X → Cut
Ctrl + P → Print

Excel:

Interface:

Workbook, Worksheet, Cells

Functions:

SUM, AVERAGE, IF, COUNTIF

Charts:

Bar, Pie, Line

Shortcut Keys:

Ctrl + C → Copy
Ctrl + V → Paste
Ctrl + X → Cut
Ctrl + Z → Undo

Exam Trick: MCQs often ask shortcuts + function uses + chart types.

4. Operating Systems

Definition:

Software managing hardware & software resources

Functions:

  • Memory Management
  • Process Management
  • File Management
  • Device Management

Types:

Batch OS

Multitasking OS

Real-time OS

Distributed OS

Mobile OS:

Android, iOS

Exam Trick: Memory management + multitasking examples are common PYQs.

5. Cybersecurity Basics

Computer Virus:

Types: Trojan, Worm, Ransomware

Symptoms: Slow performance, pop-ups, crashes

Firewall:

Hardware/Software

NGFW, Packet Filtering, Proxy

Cybercrime:

Phishing

Malware

Identity Theft

Hacking

Safe Practices:

2FA, regular updates, avoid unknown links

Exam Trick: Remember "Phish → Hook, Malware → Harm, Firewall → Guard".

6. Acronyms Quick Sheet

Acronym Meaning Use/Example
CPU Central Processing Unit Brain of the computer
RAM Random Access Memory Temporary storage
ROM Read-Only Memory Permanent storage
BIOS Basic Input Output System Booting
LAN Local Area Network Small network
WAN Wide Area Network Large network
IP Internet Protocol Unique address
DNS Domain Name System Converts domain to IP
HTML HyperText Markup Language Website code
HTTP HyperText Transfer Protocol Communication on web
URL Uniform Resource Locator Website address
PDF Portable Document Format Document format
SSD Solid State Drive Storage device
HDD Hard Disk Drive Storage device
USB Universal Serial Bus Connects devices

Exam Trick: Acronyms = favorite PYQ topic. Memorize table for fast answers.

7. Quick Revision Tips

Make flashcards for acronyms and definitions

Practice shortcuts in Word & Excel regularly

Revise PYQs for each topic before exam

Remember the IPOS cycle: Input → Processing → Output → Storage

Use the "Trick Table" method for malware, firewall, phishing

Final Exam Tip

Delhi Police Computer section mostly asks definitions, functions, types, acronyms, and shortcuts. Quick revision notes + PYQs + tricks = high chance of scoring full marks.
