Common Computer Acronyms

Delhi Police Exams

Acronyms are among the most repeatedly asked topics in Delhi Police, SSC, and Defence exams. They look simple — but one wrong word can change the meaning. Let's master them smartly.

Common Acronyms and Their Full Forms

Acronym Full Form Function
CPU Central Processing Unit Brain of computer; processes all instructions
RAM Random Access Memory Temporary memory for active data
ROM Read Only Memory Non-volatile memory storing BIOS & startup data
BIOS Basic Input Output System Loads OS from storage to memory during boot
HDD Hard Disk Drive Permanent storage for data
SSD Solid State Drive Faster storage device (no moving parts)
USB Universal Serial Bus Interface for external devices
LAN Local Area Network Connects computers within limited area
WAN Wide Area Network Connects computers across large regions
HTTP Hyper Text Transfer Protocol Protocol for web data transfer

Tip: Focus on understanding the function along with the full form for better retention
