Storage & Hardware Acronyms

Delhi Police Exams

Understanding storage and hardware acronyms is crucial for the Delhi Police Computer Knowledge section. These terms form the foundation of computer fundamentals and frequently appear in MCQs. Let's explore the essential hardware acronyms that every aspirant must know.

Common Storage & Hardware Acronyms

Acronym Full Form Meaning / Use
SSD Solid State Drive Faster storage device using flash memory
HDD Hard Disk Drive Traditional magnetic storage device
USB Universal Serial Bus Standard interface to connect devices
RAM Random Access Memory Temporary memory for running programs
ROM Read Only Memory Permanent memory storing essential instructions
CD Compact Disc Optical storage medium for data/music
DVD Digital Versatile Disc Higher capacity optical storage
Blu-ray Blu-ray Disc High-capacity optical storage for HD video
SD Card Secure Digital Card Portable flash storage for cameras, phones
NIC Network Interface Card Hardware that connects a computer to a network

Tip: Focus on understanding the practical differences between storage types and hardware components

Practice Questions (Delhi Police PYQ-style)

1

What is the full form of SSD?

Options: (a) Solid State Disk (b) Solid State Drive (c) Single Storage Drive (d) System Storage Device

Category: Storage Devices

Show Answer

Solid State Drive

2

Which of these is a traditional magnetic storage device?

Options: (a) SSD (b) HDD (c) USB (d) RAM

Category: Storage Types

Show Answer

HDD

3

RAM is used for:

Options: (a) Permanent storage (b) Temporary storage while programs run (c) Storing images only (d) Storing BIOS settings

Category: Memory Types

Show Answer

Temporary storage while programs run

Quick Recap & Tricks

Key Comparisons:

SSD > HDD in speed

RAM is temporary, ROM/HDD is permanent

GPU = graphics, CPU = brain

Memory Trick:

"Some Happy Users Really Need Good Central Power Blocks"

SSD → HDD → USB → RAM → NIC → GPU → CPU → PSU → BIOS

USB = connector

NIC = network access

BIOS = starts computer

PSU = powers computer

CD/DVD/Blu-ray/SD = portable storage
