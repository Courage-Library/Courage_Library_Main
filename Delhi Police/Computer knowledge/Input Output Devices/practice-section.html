Delhi Police Exam

Introduction to Computer Working Principle

This practice section is Delhi Police-ready, combining diagram-based questions and MCQs to strengthen your understanding of the working principle of a computer.

Exam Insight: Diagram-based questions are very common in Delhi Police exams. Focus on data flow, CPU components, and memory types.

1. Diagram-Based Questions

1

Draw a simple block diagram showing the flow of data in a computer

Hint: Include Input → CPU (ALU + Control Unit) → Output → Storage

View Answer
User → Input Device → CPU
                ├─ ALU (Arithmetic & Logic Unit)
                └─ Control Unit
         → Output Device → Storage → User
2

Label the components in the following diagram:

Imagine a simple diagram with CPU, RAM, Input/Output devices, Storage

View Answer
  • CPU → Central Processing Unit
  • ALU → Arithmetic Logic Unit inside CPU
  • Control Unit → inside CPU
  • RAM → Temporary Storage
  • Input Devices → Keyboard, Mouse, Scanner
  • Output Devices → Monitor, Printer
  • Storage → HDD/SSD
3

Identify which unit temporarily holds data during processing

View Answer

RAM (Random Access Memory)

4

In the block diagram, where does arithmetic and logical operation occur?

View Answer

ALU (Arithmetic Logic Unit)

5

Indicate the flow of data from a user entering input on a keyboard to the output on a monitor

View Answer

Keyboard → Input Unit → CPU (ALU + Control Unit) → Output Unit (Monitor)

2. MCQs (Delhi Police PYQ-style)

1

Which unit of a computer is called the brain?

Options: (a) RAM (b) CPU (c) HDD (d) Keyboard

View Answer

CPU

2

Temporary memory used during processing is called:

Options: (a) SSD (b) RAM (c) ROM (d) Printer

View Answer

RAM

3

Which CPU component performs arithmetic and logic operations?

Options: (a) Control Unit (b) ALU (c) Monitor (d) RAM

View Answer

ALU

4

Permanent storage devices include:

Options: (a) RAM (b) SSD/HDD (c) Keyboard (d) Mouse

View Answer

SSD/HDD

5

Control Unit is responsible for:

Options: (a) Processing calculations (b) Controlling all operations of the computer (c) Temporary storage (d) Displaying output

View Answer

Controlling all operations of the computer

6

Which of the following is an input device?

Options: (a) Printer (b) Mouse (c) Monitor (d) Speaker

View Answer

Mouse

7

Which of the following is an output device?

Options: (a) Keyboard (b) Scanner (c) Speaker (d) Mouse

View Answer

Speaker

8

Data flow in a computer is:

Options: (a) Output → CPU → Input → Storage (b) Input → CPU → Output → Storage (c) CPU → RAM → Input → Output (d) RAM → CPU → Monitor → Input

View Answer

Input → CPU → Output → Storage

9

ALU and Control Unit together form:

Options: (a) Input Unit (b) Output Unit (c) CPU (d) Storage Unit

View Answer

CPU

10

Which of the following devices converts user commands into digital data?

Options: (a) Input Device (b) Output Device (c) Storage Device (d) Monitor

View Answer

Input Device

Quick Recap & Memory Tricks

Data Flow: Input → CPU → Output → Storage

CPU Components: ALU + Control Unit

ALU Function: Arithmetic & Logic operations

Control Unit: Controls all computer operations

Memory Types: RAM → Temporary, HDD/SSD → Permanent

Memory Trick: "Input enters, CPU processes, Output shows, Storage keeps"

Exam Tip: MCQs are usually one-liners derived from the diagram itself. Practice drawing the basic computer block diagram multiple times.
