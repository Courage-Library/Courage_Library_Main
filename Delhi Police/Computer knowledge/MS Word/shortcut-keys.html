Shortcut Keys

Delhi Police Exam

1. Introduction to Shortcut Keys

Shortcut keys are combinations of keys that perform specific tasks quickly without navigating the Ribbon.

Saves Time

Quick access to functions

Increases Productivity

Work more efficiently

Reduces Mouse Clicks

Less repetitive actions

Common Formats:

Ctrl + Key

Alt + Key

Shift + Key

4. Quick Recap

Copy / Cut / Paste: Ctrl+C, Ctrl+X, Ctrl+V

Bold / Italic / Underline: Ctrl+B, Ctrl+I, Ctrl+U

Undo / Redo: Ctrl+Z, Ctrl+Y

Select All: Ctrl+A

Find / Replace: Ctrl+F, Ctrl+H

Hyperlink: Ctrl+K

Spell Check: F7

Alignment: Ctrl+E / Ctrl+L / Ctrl+R / Ctrl+J
