Formatting Text, Paragraph, Page Setup

Delhi Police Exam

1. Formatting Text

Text formatting is used to make documents readable, attractive, and professional.

Key Features:

Font Style & Size → Change appearance and size of text

Bold, Italic, Underline → Emphasize text

Font Color & Highlighting → Different colors for emphasis

Text Effects & Typography → Shadow, glow, reflection

Useful Shortcuts:

Change Font

Ctrl + Shift + F

Increase Size

Ctrl + Shift + >

Decrease Size

Ctrl + Shift + << /p>

Bold

Ctrl + B

2. Paragraph Formatting

Paragraph formatting controls spacing, alignment, and structure of text blocks.

Feature Description
Alignment Left, Right, Center, Justify
Line Spacing Single, 1.5 lines, Double
Indentation First line indent, Hanging indent, Left/Right indent
Bullets & Numbering Organize lists clearly
Borders & Shading Add borders/background color to paragraph

Shortcut Tips:

Center Align

Ctrl + E

Left Align

Ctrl + L

Right Align

Ctrl + R

Justify

Ctrl + J

3. Page Setup

Page setup controls the layout and structure of the document.

Option Purpose
Margins Set space between text and page edge
Orientation Portrait / Landscape
Size A4, Letter, Legal, Custom sizes
Columns Split page into multiple columns
Line Numbers Add line numbers for reference
Breaks Page Break, Section Break for formatting

Useful Tip: Page setup is important before printing to maintain professional appearance.

5. Quick Recap

Text Formatting → Font, Size, Color, Bold, Italic, Underline

Paragraph Formatting → Alignment, Indentation, Line Spacing, Bullets/Numbering

Page Setup → Margins, Orientation, Size, Columns, Breaks

Frequently Asked Shortcuts: Ctrl+B, Ctrl+I, Ctrl+U, Ctrl+E, Ctrl+J

You've completed MS Word Formatting Concepts!

Courage Tip: Master the keyboard shortcuts for formatting - they save time and are frequently tested. Remember that text formatting affects individual characters, paragraph formatting affects entire paragraphs, and page setup affects the entire document layout.
