Slide Transitions and Animations

Delhi Police Exam - Computer Knowledge

1. Introduction

PowerPoint makes presentations more dynamic and visually engaging using Transitions and Animations. Both add movement to slides â€” but they work differently.

Transition

Effect that occurs when moving from one slide to another

Animation

Effect that occurs within a single slide to highlight objects

2. What are Slide Transitions?

Slide Transitions control how one slide enters or exits during the slideshow. They are applied between slides to create smooth or dramatic visual changes.

Purpose Change slide appearance while shifting Fade, Push, Wipe

3. What are Animations?

Animations make objects move, appear, or disappear on a slide. They help in focusing audience attention and explaining step-by-step concepts.

Feature Description Example
Purpose Add motion to text or images Fly In, Appear, Zoom
Apply From Animations Tab Select object â†’ Choose effect
Types of Effects 4 types: Entrance, Emphasis, Exit, Motion Path Explained below
Preview View animation before applying "Preview" button in Animations tab

4. Types of Animation Effects

Type Description Example
Entrance Object appears on slide Appear, Fly In, Fade
Emphasis Highlights or changes object already on slide Spin, Grow/Shrink
Exit Object disappears from slide Fly Out, Fade Out
Motion Path Object moves along a custom path Circle, Line, Curve
